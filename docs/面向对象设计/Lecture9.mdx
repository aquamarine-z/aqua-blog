# Lecture 9

## 关键概念中英文对照及解释

1. 评估方法 (Evaluation Approaches)
   - **中文**：评估方法
   - **英文**：Evaluation Approaches
   - **解释**：用于确定设计或原型有效性的方法，包括可用性测试、实验和现场研究等。

2. 数据收集技术 (Technologies for Data Collection)
   - **中文**：数据收集技术
   - **英文**：Technologies for Data Collection
   - **解释**：在进行评估时使用的技术和工具，如移动应用、传感器、眼动追踪设备等。

3. 伦理考虑 (Ethical Considerations)
   - **中文**：伦理考虑
   - **英文**：Ethical Considerations
   - **解释**：在评估过程中需要考虑的伦理问题，包括参与者的尊重、数据保密性和诚实报告研究结果。

## 评估方法类型

1. 可用性测试 (Usability Testing)
   - **中文**：可用性测试
   - **英文**：Usability Testing
   - **解释**：帮助测试设计原型的可用性，如效率、实用性、易学性和易用性等。

2. 实验 (Conducting Experiments)
   - **中文**：实验
   - **英文**：Conducting Experiments
   - **解释**：帮助比较两个系统，如触摸输入与键盘输入的比较。

3. 现场测试 (Field Testing)
   - **中文**：现场测试
   - **英文**：Field Testing
   - **解释**：帮助理解系统在现实世界中的应用，如用户体验、有效性或适用性。

## 课程内容

- **评估方法概览**：介绍了不同的评估方法，包括它们的定义、应用场景和执行步骤。
- **评估方法的应用**：探讨了如何将这些评估方法应用于实际案例中，如墨尔本市议会开发的停车应用测试。
- **数据收集**：讨论了在评估过程中收集数据的重要性和常用的数据收集方法，如访谈、问卷、观察等。
- **伦理考虑**：强调了在评估过程中需要考虑的伦理问题，包括研究设计的伦理问题、数据收集的伦理问题以及数据分析和报告的伦理问题。
