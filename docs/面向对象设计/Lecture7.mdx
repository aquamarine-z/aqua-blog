# Lecture 7

## 关键概念中英文对照及解释

1. 交互技术 (Interaction Technologies)
   - **中文**：交互技术
   - **英文**：Interaction Technologies
   - **解释**：指用户与系统之间进行交互的各种技术，如命令行界面、图形用户界面等。

2. 界面 (Interface)
   - **中文**：界面
   - **英文**：Interface
   - **解释**：用户与系统之间交流任务和结果的媒介。

3. 交互风格 (Interaction Styles)
   - **中文**：交互风格
   - **英文**：Interaction Styles
   - **解释**：从系统的角度描述交互的方式，如指令式、对话式、操作式等。

4. 输入设备 (Input Devices)
   - **中文**：输入设备
   - **英文**：Input Devices
   - **解释**：用户向系统提供输入的设备，如键盘、鼠标、触摸屏等。

## 界面类型

1. 命令界面 (Command Interface)
   - **中文**：命令界面
   - **英文**：Command Interface
   - **解释**：通过命令语言进行交互的界面。

2. 图形界面 (Graphical Interface)
   - **中文**：图形界面
   - **英文**：Graphical Interface
   - **解释**：用户通过图形元素进行交互的界面。

3. 多媒体界面 (Multimedia Interface)
   - **中文**：多媒体界面
   - **英文**：Multimedia Interface
   - **解释**：使用文本、图像、视频等多种媒介进行信息交流的界面。

4. 虚拟现实界面 (Virtual Reality Interface)
   - **中文**：虚拟现实界面
   - **英文**：Virtual Reality Interface
   - **解释**：用户在虚拟世界中探索现实环境的界面。

5. 网站界面 (Web Interface)
   - **中文**：网站界面
   - **英文**：Web Interface
   - **解释**：通过网站进行交互的界面。

6. 移动界面 (Mobile Interface)
   - **中文**：移动界面
   - **英文**：Mobile Interface
   - **解释**：在移动设备上进行交互的界面。

7. 家电界面 (Appliance Interface)
   - **中文**：家电界面
   - **英文**：Appliance Interface
   - **解释**：家用设备上用于特定目的的界面。

8. 语音界面 (Voice Interface)
   - **中文**：语音界面
   - **英文**：Voice Interface
   - **解释**：通过语音进行交互的界面。

9. 笔式设备界面 (Pen Based Devices Interface)
   - **中文**：笔式设备界面
   - **英文**：Pen Based Devices Interface
   - **解释**：使用笔和触控笔在触屏上进行交互的界面。

10. 触摸屏界面 (Touchscreen Interface)
    - **中文**：触摸屏界面
    - **英文**：Touchscreen Interface
    - **解释**：通过触摸屏幕进行交互的界面。

## 课程内容

- **交互技术回顾**：回顾了上周讨论的交互技术，包括命令行界面、图形用户界面、自然用户界面等。
- **界面类型**：介绍了不同类型的界面，如命令界面、图形界面、多媒体界面等，并讨论了它们的特点和适用场景。
- **界面设计因素**：探讨了影响界面设计选择的因素，如技术暴露、特定需求、成本等。
- **界面设计的艺术与科学**：讨论了界面设计是否更多地与艺术或科学相关，以及为什么。
