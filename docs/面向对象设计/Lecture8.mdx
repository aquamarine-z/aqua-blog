# Lecture 8
## 关键概念中英文对照及解释

1. 多模态界面 (Multi-modal Interface)
   - **中文**：多模态界面
   - **英文**：Multi-modal Interface
   - **解释**：使用触摸、声音、视觉和语言等多种方式呈现信息的界面，旨在提供类似现实世界的交互体验。

2. 可共享界面 (Shareable Interface)
   - **中文**：可共享界面
   - **英文**：Shareable Interface
   - **解释**：为协作操作而设计的界面，能够同时接收多个输入，如多点触控桌面。

3. 有形用户界面 (Tangible User Interface, TUI)
   - **中文**：有形用户界面
   - **英文**：Tangible User Interface
   - **解释**：用户在实体对象或与之关联的数字系统中看到反馈的界面。

4. 增强现实界面 (Augmented Reality Interface, AR)
   - **中文**：增强现实界面
   - **英文**：Augmented Reality Interface
   - **解释**：通过数字元素增强现实世界环境的界面。

5. 可穿戴界面 (Wearables Interface)
   - **中文**：可穿戴界面
   - **英文**：Wearables Interface
   - **解释**：穿戴在身上的智能设备界面，如智能手表和健康追踪器。

6. 机器人和无人机界面 (Robots and Drones Interface)
   - **中文**：机器人和无人机界面
   - **英文**：Robots and Drones Interface
   - **解释**：用于家庭清洁、园艺以及伴侣机器人的界面，以及远程控制的无人机界面。

7. 智能界面 (Smart Interfaces)
   - **中文**：智能界面
   - **英文**：Smart Interfaces
   - **解释**：能够感知周围环境并执行适当操作的上下文感知界面。

## 课程内容

- **界面类型概览**：回顾了上周讨论的界面类型，并继续探讨剩余的界面类型。
- **界面设计的应用**：讨论了不同界面类型在教育实践、智能家居、手术等领域的应用。
- **界面与交互风格的映射**：将不同的界面类型与交互风格相映射，以更好地理解它们的使用场景。
- **未来界面设计**：探讨了界面设计的未来趋势，以及可能获得更多关注和流行的界面类型。
