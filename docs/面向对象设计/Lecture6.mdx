# Lecture 6

1. 交互设计范式 (Interaction Design Paradigm)
   - **中文**：交互设计范式
   - **英文**：Interaction Design Paradigm
   - **解释**：指交互设计中的基本理念和方法，包括用户与系统之间的交互方式和设计原则。

2. 输入与输出 (Input & Output)
   - **中文**：输入与输出
   - **英文**：Input & Output
   - **解释**：用户与系统交互过程中的输入（如键盘、鼠标操作）和系统对用户操作的输出响应。

3. 命令行界面 (Command Line Interface, CLI)
   - **中文**：命令行界面
   - **英文**：Command Line Interface
   - **解释**：基于文本的界面，用户通过输入命令行来操作计算机。

4. 图形用户界面 (Graphical User Interface, GUI)
   - **中文**：图形用户界面
   - **英文**：Graphical User Interface
   - **解释**：使用图形图标（如光标、窗口等）与系统交互的方式。

5. 自然用户界面 (Natural User Interface, NUI)
   - **中文**：自然用户界面
   - **英文**：Natural User Interface
   - **解释**：利用人类自然能力（如注视、触摸、语音、手势、身体动作等）与系统交互的界面。

6. 有形用户界面 (Tangible User Interface, TUI)
   - **中文**：有形用户界面
   - **英文**：Tangible User Interface
   - **解释**：支持物理操作的界面，通过实体对象与数字信息的结合来实现交互。

7. 脑机接口 (Brain Computer Interface, BCI)
   - **中文**：脑机接口
   - **英文**：Brain Computer Interface
   - **解释**：使人类能够通过思维控制机器的接口，可以是直接连接到大脑组织或作为可穿戴设备放置在用户头部。

8. 人类食物交互 (Human Food Interaction, HFI)
   - **中文**：人类食物交互
   - **英文**：Human Food Interaction
   - **解释**：结合触觉、味觉和嗅觉来创造多感官界面的新兴领域。

9. 动物计算机接口 (Animal Computer Interface, ACI)
   - **中文**：动物计算机接口
   - **英文**：Animal Computer Interface
   - **解释**：不仅为动物设计，主要满足动物需求的技术，用于与人类互动、理解动物行为和动物娱乐。

## 课程内容

- **交互设计范式**：讨论了不同交互设计范式及其对用户体验的影响。
- **输入与输出**：探讨了用户与系统之间交互的基本组成部分。
- **交互技术的历史与现状**：从命令行界面到图形用户界面，再到自然用户界面等的发展。
- **交互技术的关键特征与优势**：分析了不同交互技术的主要特点和它们相对于其他技术的优势。
- **交互技术的选择**：讨论了选择合适交互技术时需要考虑的因素，如用户群体、成本、硬件限制等。
