# N2语法学习笔记Day8

## 71.〜にて
### 释义 
中 : 在……    
日 : 〜で  
### 接续
名词 + にて  
### 例句
本日のイベントは1階の広場にて行います。  
今天的活动在1楼的广场举行。  

明日は金時計にて集合しましょう。  
明天在金钟见面吧。  

当店は午前9時にて開店いたします。  
我们店早上 9 点开门。  
### 笔记
・主要以书面形式出现，或在公告中看到  
・强调的方法、时间、地点、原因或起源  

## 72.〜のみならず
### 释义 
中 : 不仅……，此外还有其他的。  
日 : ～だけでなく、ほかにもある。
### 接续
动词普通形 + のみならず  
名词 +（である）+ のみならず  
形容词辞书形 + のみならず  
形容动词词干 +（である）+ のみならず 
### 例句
与党のみならず、野党も党首の選出には慎重だ。  
不仅仅是执政党，在野党的党首选举也很慎重。  

石井さんは、差別的な発言をしたのみならず、それについて謝ることもなかった。  
石井非但做了歧视性的发言，而且也没就此道歉。  
### 笔记
不同于「～に限らず」，也可以用于同等级的其他事物也一样的情况。    
后面多是含有“其他也”含义的使用「も」的句子。  
本身 のみ 就是仅限的意思 而这个语法相当于 のみ + なる的ず形表否定 所以就是不仅的意思
书面语。    

## 73.〜っぱなし
### 释义 
中 : 一直……；就这样……  
日 : 〜したままだ
### 接续
动词连用形 + っぱなし  
### 例句
家に帰ると、妻がテレビをつけっぱなしで寝ていた。  
回到家后，看到妻子电视还开着就睡着了。  

後半になって我がチームはずっと相手に押されっぱなしになっていた。  
到了下半场，我队一直被对手压着打。  
### 笔记
・一直持续着某种放任状态。用于该做却没做的事情上。  
・带有批评、埋怨的语气。  

## 74.〜ぶる
### 释义 
中 : 装作……的样子。  
日 : 本当は～という素質や性質を持っていないのに、それを持っているような態度をすることです。  
### 接续
名词 ＋ ぶる  
形容词词干 ＋ ぶる  
形容动词词干 ＋ ぶる  
### 例句
彼は金持ちぶってるが、本当は借金がたくさんある。   
他装着是有钱人，其实有很多欠款。  

息子は悪ぶっているが、実は気の弱い優しい子です。  
儿子装着很坏，实际上是胆子小性情柔弱的孩子。    
### 笔记
实际上并非如此,而是好像是某种态度、装作某种样子。    
多用于说话人对某人的负面影响。    
<b>意思与「~ふりをする」相同。「~ぶる」较强地表现说话者厌恶的感觉。</b>  
<b>・带有批评、埋怨的语气。  </b>  

## 75.〜ないことには（〜ない）
### 释义 
中 : 如果不……某事就无法实现。  
日 : ～なければ、あることが実現しない。
### 接续
动词ない形去ない ＋ ないことには  
形容词く形 ＋ ないことには  
形容动词词干 + で ＋ ないことには  
名词 + で ＋ ないことには  
### 例句
一度会ってみないことには、仕事を任せられる人かどうかわからない。   
如果不见一次面，就不知道对方是不是可以胜任工作的人。    

足がもっと丈夫でないことには、あの山に登るのは無理だろう。     
如果腿脚不更结实一点的话，登那座山很困难吧。     
### 笔记
后续接否定含义的句子。表示必要条件，表达说话人否定、消极的感情色彩。  

## 76.〜ないことはない・〜ないこともない
### 释义 
中 : 有……的可能；不是绝对不……  
日 : 〜という可能性があるかもしれない / 絶対に〜とは言えない
### 接续
动词ない形 + ことはない・こともない  
形容词否定形 + ことはない・こともない  
形容动词词干 + ではない + ことはない・こともない  
名词 + でない + ことはない・こともない  
### 例句
できないこともないですけど、少し時間をいただきたいです。  
也不是做不到，只是想要多一点点时间。  

映画は見ないことはないが、詳しくもない。  
也不是不看电影，只是对电影了解不多。  

元気ではないことはないけど、すごく元気なわけでもない。  
虽然不是没有精神，但是也不是特别有精神。  

吉田：「あなたは誕生日ケーキをカットしてみたい？」山田：「僕にもできないことはないけど、君がやった方がいいと思う。」  
吉田：“你想试试切生日蛋糕吗？” 山田：“虽然也不是做不到，但是我觉得你来切会更好。”  
### 笔记
避免断定，委婉的表达方式。  

## 77.〜ないで済む・〜ずに済む
### 释义 
中 : 不用……就解决了。  
日 : 〜しなくてもいい / 〜なくても問題はない。
### 接续
动词ない形去ない + ないで済む・なくて済む・ずに済む  
名词 + なしで済む  
### 例句
お店の商品を壊してしまったが、必死に謝ったらお金を払わずに済んだ。  
虽然弄坏了店里的商品，但拼命道歉了之后，没有赔钱就解决了。    
### 笔记
表示在大多数情况下，不用再做原来预定要做的不愉快或麻烦的事情，因此强调说话者因可以避免它而感到宽慰。书面语。  
口语可说成「ないですむ」。  
否定形式为「ずにはすまない」，译为“必须，一定”。  
<b>注意名词的接续形式!</b>

## 78.〜ないでもない・〜ないものでもない
### 释义 
中 : 也不是不…  
日 : 全く〜ないわけではない / 場合によっては〜する
### 接续
动词ない形去ない + ないでもない・ないものでもない  
形容词く形 + ないでもない・ないものでもない  
形容动词词干 + で + ないでもない・ないものでもない  
名词 + で + ないでもない・ないものでもない  
### 例句
「なんとなく彼が言った話には欠陥や問題がたくさんあるような気がしないでもないが、いってくれないよりマシだ。」  
“虽然我也不是没有察觉他跟我说的话有很多漏洞和问题，但总比不告诉我好。” 

駅まで走っていけば、終電に間に合わないものでもない。  
跑着去电车站的话，也不是赶不上末班车。  
### 笔记
・意思与「〜なくもない」相同。用双重否定表示消极的肯定。  
<b>・多用于表达个人的判断、推测、爱憎等。</b>

## 79.〜ながら（も）
### 释义 
中 : 尽管……却……  
日 : 〜のに / 〜けれども
### 接续
动词连用形 + ながら(も)  
形容词辞书形 + ながら(も)  
形容动词词干 + ながら(も)  
名词 + ながら(も)  
### 例句
この地域の人々は貧しいながらも、幸せに暮しているそうです。  
虽然这里的人们很穷，但看来他们生活得很幸福。  

敵ながら見事な戦術だ。  
即使是敌人，这也是一个绝妙的战术。  
### 笔记
表示如果在通常情况下，前期的事实不会导致后项这种的结果，然而却出现了这种结果。  
表达讲话人一种强烈的惊讶、不满、谴责或深表遗憾的心情。有时也用于谨慎、谦让的表达中。    

## 80.〜なんて
### 释义 
中 : ……什么的  
日 : 〜ということを
### 接续
动词简体形 + なんて  
形容词简体形 + なんて  
形容动词词干 + だ + なんて  
名词 + だ + なんて  
### 例句
まさか、あの二人が離婚だなんて。信じられない。  
怎么会，那两人竟然离婚了。不敢相信。  

約束を忘れるなんて、メアリーの彼氏酷すぎだよ。  
忘记约定什么的，梅亚里的男朋友太过分了。  

まさかこんな形で再会するなんて思いもしなかった。  
没想到我们会以这样的方式再遇见。  
### 笔记
看到或听说意想不到的事实时，表惊讶或感慨的说法
