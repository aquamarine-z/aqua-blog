# N2语法学习笔记Day5

## 41.〜そうにない・〜そうもない
### 释义
中 : 能做到……的可能性很低  
日 : 〜できる可能性が低いことを表す。
### 接续
动词连用形 + そうにない・そうもない  
动词可能形去る + そうにない・そうもない  
### 例句
すみません、今日中に終わりそうにないんですが、明日でもよろしいでしょうか。  
抱歉，今天内好像不太能完成，可以明天再做吗？  

ごめん、定時で上がれそうにないから先に食べてて。  
抱歉，我好像不太可能准时下班了，你先吃吧。   
### 笔记
体现了一种主观的判断或推测，多用于消极的场合。  
这两个接续中的 动词可能形去る + そうにない・そうもない  也就是可能形连用形 + そうにない・そうもない  因此可以归为一类?  

## 42.〜だけあって・〜だけのことはあって
### 释义
中 : 与……相称；不愧是……  
日 : 〜ので、当然・・・だ。  
### 接续
动词(普通形) + だけ（のことは）あって  
形容词简体形 + だけ（のことは）あって  
形容动词词干 + な + だけ（のことは）あって  
名词 + だけ（のことは）あって  
### 例句
あの二人は仲良しなだけあって、いつも一緒にいるよね。    
那两人不愧是好朋友，总是在一起呢。    

彼は10年間日本に住んでいただけあって、日本語を日本人のように話せる。    
他不愧在日本住了10年，说日语像日本人一样流利。    

茶道を習っているだけあって、礼儀が正しい    
不愧是修习茶道之人，举止端庄。    

あのホテルは高いだけあって、サービスがとてもいい。    
不愧那家酒店价格昂贵，服务非常好。    
### 笔记
表示与其身份，地位，经历等相称，后项对其进行正面的评价。    
常与「さすがに」呼应使用。    
由于「だけあって」强调的是固有或恒定性质，因此它不会用于不确定或尚未发生的事情。  
<b>这意味着「だけあって」不会出现在讨论将来可能发生的事情的句子中，也不会经常与だろう、でしょう、かもしれない或类似表达不确定性的结构一起使用。</b>  
だけ的成分跟名词相似 所以和名词接续相似

## 43.〜だけに
### 释义
中 : 正因为……（所以）更加……。  
日 : 〜ので、もっと・・・だ。 / 〜ので、なおさら・・・だ。   
### 接续
动词简体形 + だけに  
形容词简体形 + だけに  
形容动词词干 + な/である/ だった + だけに  
名词 + （な / である/ だった） + だけに  
### 例句
初めての海外旅行なだけに、かなり心配だ。  
正因为是第一次出国旅行，所以我非常担心。  

このアパートは東京駅に近いだけに、家賃が高い。  
正因为这间公寓离东京站近，所以租金更贵了。  

このアプリは便利なだけに、どんどんとユーザーが増えてきています。  
正因为这个软件非常方便，所以用户数量正在迅速增加。  
### 笔记
由于……而产生的结果，在突出产生积极或消极的结果时使用  
<b>注意名词的接续中（な / である/ だった）是可以去掉的</b>

## 44.〜だけましだ
### 释义
中 : 只有……是好的。  
日 : 今の状況は良くないが、〜よりはいい。
### 接续
动词简体形+ だけましだ  
形容词简体形+ だけましだ  
形容动词词干 + な + だけましだ  
名词 + である + だけましだ  
### 例句
今日はいつもより寒いけど、風が吹いていないだけまだましだ。  
今天虽然比平时冷，但至少没有刮风。    

このアパートはボロボロだけど、職場に近いだけましだ。  
这间公寓虽然很破旧，但至少离我工作的地方很近。  

あの講師の教え方は上手とは言えないが、黒板の字が綺麗なだけましだ。  
虽然那个老师的教学方式算不上好，但至少黑板字写得工整。  
### 笔记
不幸中的万幸。  
<b>注意名词的接续需要添加 である </b> 
在 「〜だけましだ」 中，「だけ」 的作用是限定一个相对较好的方面，强调“还算是……”的意思。   
「まし」 是形容动词，表示“稍微好一点，聊胜于无”。    
在 「〜だけましだ」 中，「ましだ」 的作用是表达对现状的评价，认为虽然并不完美，但相较于更糟的情况已经是好的了。 

## 45.〜たて
### 释义
中 : 刚刚……完  
日 : 〜したばかりで
### 接续
动词连用形 + たて   
动词连用形 + たて + の + 名词  
### 例句
やっぱり料理はできたてが美味しいね。  
果然刚做好的饭是最好吃的。 

焼きたてのパン。  
新鲜出炉的面包。  
### 笔记
「たて」是一个后缀，它用来表达某件事完成后的那一刻，  
而不是它“已经”完成的事实。因此，「たて」最常与表示某种基于时间的重要性的动词一起使用，  
<b>其中修饰的事物在完成后会随着时间的推移而质量下降。</b>  

## 46.〜たものではない
### 释义
中 : 太……而不能……  
日 : とても〜できない
### 接续
动词た形 ＋ ものではない  
### 例句
このジュースは健康に良いようで、最近人気があるが、酸っぱすぎて飲めたものではない。  
这个果汁好像对健康有益，最近也很有人气，但实在是太酸了，根本喝不下去。  

大雑把な彼に旅行の計画を任せたらどうなるか、わかったものではない。  
如果把旅游计划交给做事那么随便的他的话，不能想象会变成什么样。  
### 笔记
<b>用于负面评价</b>  

## 47.〜っこない
### 释义
中 : 根本不可能……；绝对不会……。  
日 : 絶対に～ない / 〜はずがない。  
### 接续
动词连用形 ＋ っこない  
### 例句
今日は雨が降ってるし、風邪が強いから野球なんてできっこない。  
今天又是雨天，感冒也很严重，肯定打不了棒球了。  
### 笔记
<b>表示否定，对某事不看好，对某件事发生的可能性进行强烈的否定。</b>    

## 48.〜つつ（も）
### 释义
中 : 虽然……，但是……。  
日 : ～ているけれども。
### 接续
动词连用形 + つつ（も）   
### 例句
初めてのライブで緊張しつつもものすごく楽しめました。  
因为这是我第一次开演唱会，我很紧张，但是也非常非常开心！  

南極点遠征はよく備えつつも、失敗した。  
尽管准备充分了，南极探险还是失败了。  
### 笔记
用于连接前后两个相反的事项。用于“言行不一”，“想的和做的是两码事”等语言环境中。  
同 ～のに 的意思基本相同，<b>书面语</b>。用于表达说话人的反省、后悔、谴责等语气。   

## 49.〜つつ
### 释义
中 : 一边……，一边……  
日 : 〜ながら  
### 接续
动词连用形 + つつ（も）   
### 例句
京都にある「哲学の道」を歩きつつ、今後の人生について考えてみる。    
我一边漫步于京都的“哲学之道”，一边思考着往后的人生。  

インターネットでレシピを見つつ、料理をしたので今日の晩御飯は美味しいはずです。  
这是我一边看网上的菜谱一边做的，今天的晚饭应该会很好吃。  
### 笔记
常用于动作持续状态，而不是涉及运动的实际物理动作。  
「つつ」 常用于抽象行为（如思考、努力等）或强调动作的持续状态，而不适用于描述实际的、具体的物理运动动作。  
对于涉及物理运动的动作，建议用更自然的 「ながら」。  

## 50.〜つつある
### 释义
中 : 渐渐……。  
日 : だんだん～ている。今まさに～ている。
### 接续
动词连用形 + つつある。  
### 例句
事故の原因は明らかになりつつある。    
事故的原因渐渐明晰。    
### 笔记
表示动作正在朝着完成的方向进行。用于书面语。  

## 总结
跟 つつ 有关的前面都接动词连用形  
在日语中，「つつ」 的本义是表示动作或状态的持续，即某个行为正在发生并持续进行。  
古日语中，「つつ」常被用来表示重复或持续发生的动作，类似于现代的“反复做某事”。  
在现代日语中，「つつ」 的本义衍生出两种主要用法：  

 - 表示“动作的同时性”   
 - 表示“动作的持续性”  

同时由于 だけ 的几个文法中接续并不完全相同 所以需要特殊记忆
