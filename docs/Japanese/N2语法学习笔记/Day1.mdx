# 日语N2语法学习笔记 Day1
## 1.~あげく
### 释义
中 : 做了很多...,最后还是...。  
日 : 色々～したけど、最終的に。
### 接续
动词た形+あげく  
名词+の+あげく
### 例句
口論のあげくに殴り合いになった。  
争论到最后拳脚相向。 

どうしてよいかわからなかったが、しばらく考えたあげくにこういった。   
我不知道该怎么办，但想了一会儿，我终于说出了这句话。  

## 2.あまり（に）  
### 释义
中 : 过于......   
日 : ~あまり（に）  
### 接续
动词普通形 + あまり（に）  
动词ている形 + あまり（に）  
形容词辞书形 + あまり（に）  
形容词た形 + あまり（に）  
形容动词词干 + な+あまり（に）  
名词 + の + あまり（に）  
### 例句
田舎に引っ越したばかりのころは、寂しいあまり友達に毎日のように電話したものだ。  
刚搬到乡下的时候，我因为过于孤独，每天都会打电话给我的朋友。  　

急いでいたあまりに、大事な書類を家に忘れてきてしまった。  
我太过匆忙了，把重要的文件忘在家里了。  

## 3.〜あまりの〜に・あまりの〜で
### 释义
中 : 因为真的特别……。  
日 : とても〜すぎるので。  
### 接续
あまりの + 名词 + に  
あまりの + 名词 + で  

### 例句
あまりの面白さに、会場は終始笑いが絶えなかった。  
因为实在太有趣了，会场全场下来都充满着笑声。  

あまりの衝撃に、しばらく開いた口が塞がらなかった。  
因为过大的冲击，我一时震惊得合不拢嘴。  

## 4.〜以上（は）
### 释义
中 : 既然……，就……。    
日 :～なのだから、当然...    
### 接续
动词简体形 + 以上（は）     
形容词词干 + である + 以上（は）  
名词 + である + 以上（は） 
### 例句
契約書にサインした以上、引き返すことはできない。  
在合同书上签名了，就不能反悔。  

会社を辞めると決めた以上、早く上司にその旨を伝えなければ。  
既然决定辞职了，就赶紧告诉老板。  

## 5.〜一方で
### 释义
中 : 某方面是……，另一方面是……。（并列）  
日 :ある面では〜だが、別の面では〜（並列）  
### 接续
动词简体形＋一方で   
形容词简体形＋一方で    
形容动词词干＋な/である＋一方で   
名词＋の/である＋一方で    
### 例句
彼は俳優である一方で、歌手としても活躍している。   
他在担任演员的同时，也以歌手的身份活跃。  

彼はお金持ちである一方で、積極的にボランティアやチャリティーに参加する優しい人でもある。  
他不仅是有钱人，也是积极参与志愿者活动和慈善事业的善良的人。

## 6.〜以来
### 释义
中 : 从……之后，到现在也一直……  
日 :～してから、今までずっと。  
### 接续
名词 + 以来      
动词て形 + 以来   
名词＋の/である＋以上に    
### 例句
日本に来て以来、毎日家族にメールを送っている。  
自从来了日本之后，我每天都会给家人发短信。  

入社以来、彼とは毎日のように相談し合っている。  
自从进了公司以后，每天都会跟他商量事宜。  

## 7.〜上（で）
### 释义
中 : ……之后，然后。作为准备先做......，其后再做接下来的行为。  
日 : 準備としてまず～してから、その後で次に続く行動をする。  
### 接续
动词た形+上で  
名词+の+上（で）  
 ### 例句
自分一人では決められませんので、家族と相談した上で、お返事をいたします。   
我一人无法决定，我会和家人商量之后再给您答复。    

この列車には特急券が必要です。あらかじめ特急券をお買い求めの上、ご乗車ください。    
乘坐这趟列车需要用特快车票。请您事先买好特快车票再乘车。  

## 8.〜上で
### 释义
中 : 在……方面  
日 : 〜ために / 〜のに  
### 接续
动词辞书形 + 上で  
名词 + の + 上で  
### 例句
外国人を採用する上で、私たちが見ているのは日本語能力だけではありません。    
在采用外国人方面，我们看重的不只是日语能力。      

今度の文化祭を成功させる上で必要なのはクラスの団結力だ。  
要让即将到来的学园祭成功举办，需要的是班级团结。  

## 9.〜得(う)る・〜得(え)ない
### 释义
中 : 可能……/不可能……  
日 : 〜できる / 〜できない  
### 接续
动词连用形 + 得る・得ない  
### 例句
警察官は信頼し得ないと思っている人たちが増えて来た。  
认为警察不可信的人越来越多。  

消防士は信頼し得る人です。  
消防员是能信任的人。  

## 10.〜折(おり)に
### 释义
中 : ……的时候  
日 : 〜ときに / 〜の機会に  
### 接续
动词辞书形 + 折に   
动词た形 + 折に  
名词 + の + 折に  
### 例句
次にお会いした折りに、また詳しくお話いたします。  
下次见面时，我再详细跟您说。  

仕事で上海に行ったおりに、彼を訪ねた。  
借着工作去上海的机会，我去拜访过他。
