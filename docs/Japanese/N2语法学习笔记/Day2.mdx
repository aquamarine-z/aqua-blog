# N2语法学习笔记Day2

## 11.〜かいがあって・ 〜かいもなく
### 释义
中 : 有……的效果/多亏……      
日 : 〜の効果があって / 〜のおかげで

中 : 有……的价值  
日 : 〜の価値があって
### 接续
动词辞书形 + かいがあって・かいもなく  
动词た形 + かいがあって・かいもなく  
动词连用形 + かいがあって・かいもなく  
サ变名词 + の + かいがあって・かいもなく
### 例句
遠くから時間とお金をかけて来たかいもなく、店は閉まっていて名物料理が食べられなかった。  
白费时间和金钱从大老远赶过来，那间店关门了，没能吃到那道名菜。

夜11時まで残業したかいもなく、納期を守れず謝ることになった。  
白白加班到半夜11点，结果还是没能赶上交付期限，只能道歉。

今の仕事は忙しいが、やりがいがある。  
现在的工作虽然很忙，但是有做的价值。

彼はいつも面白い反応をしてくれるので、イジリがいがある。  
因为他的反应一直很有趣，所以很有捉弄他的价值。
### 笔记
甲斐（かい）: 效果；价值；好处，用处；意义(その行為に値するだけの値打ちや効果。)  
因为かい是名词，所以接续跟名词类似。需要注意的是サ变名词 + の + かいがあって・かいもなく 　

## 12.〜限り（限界）
### 释义
中 : 到……的极限；尽量……；竭尽……。        
日 : 〜の限界まで / できるところのギリギリまで
### 接续
动词辞书形 ＋ 限り  
动词可能形 ＋ 限り  
名词 + の + 限り
### 例句
できる限りのことはしたので、たとえ失敗したとしても後悔はない。  
因为尽全力去做了，所以即使失败了也不后悔。

日が暮れる前に、行ける限りのところまで行きたい。  
我想在太阳落山之前尽可能走到最远能走到的地方。
### 笔记
限り 是一个名词形式的词，所以接续跟名词类似。  
表示最大程度/达到极限。

## 13.〜限り（条件、範囲）
### 释义
中 : 在……期间一直（保持……状态）        
日 : ～する間はずっと / ～の状態が続く間はずっと
### 接续
动词辞书形 + 限り  
动词ている形 + 限り  
名词 + である + 限り
### 例句
田舎に住んでいる限り、できる仕事はあまり多くない。  
住在乡下的期间，能做的工作就不多。

私はSEである限り、いつまでも最新の技術を勉強し続けなければならない。  
只要在SE工作的期间，我就必须不断地学习最新的技术。

体が動く限りは80歳になっても、90歳になっても何かビジネスをしたいと考えています。  
在我身体还能活动的时间里，不管是到了80岁还是90岁，我都想做点生意活。
### 笔记
辞书形 + 限り：强调<b>某个动作或条件的范围</b>，通常用于<b>表示“尽可能”或“在条件允许下”</b>。    
ている形 + 限り：<b>强调持续状态或动作</b>，表示某个状态或条件下的范围<b>持续存在</b>。

## 14.〜がたい
### 释义
中 : （心情上）很难……；难以……。  
日 : 〜するのが難しい。
### 接续
动词连用形 + がたい
### 例句
彼はいつも怖い顔をしているし、静かだし、近寄りがたいよ。  
他总是板着一张脸，沉默不说话，实在是难以接近。

彼はいつも嘘ばかりつくので、今回の話も本当かどうか信じがたい。  
因为他一直说谎，所以这次说的话也很难让人相信。

大勢の人の前で馬鹿にされて、耐えがたい屈辱を味わった。  
在一大群人面前被嘲笑，我承受了难以忍受的羞辱。

20代の男性がナイフで女性を刺したそうです。本当に許しがたい行為だ。  
据说，一名20多岁的男子用刀刺伤了该女子。这真是令人无法原谅的行为。

まさか、彼女があの事故から奇跡的に生き延びた何て信じがたい話だ。  
很难相信她在那场事故中奇迹般地生还了。
### 笔记
表示不容易实现，做起来比较难的意思。<b>多用于好的事情，偶尔也用于不好的事情</b>。

## 15.〜かと思うと・〜かと思ったら
### 释义
中 : 一……，马上……。  
日 : 〜すると、すぐに
### 接续
动词た形 + かと思うと・かと思ったら
### 例句
空がピカッと光ったかと思うと、大きな雷の音が鳴った。  
天空划过一道闪电后，马上就听到了很大的打雷的声音。

空が急に暗くなってきたかと思うと、雨が降ってきた。  
天空突然暗下来不久后，雨下了起来。

息子が帰って来たかと思うと、すぐに家を出ていった。
正当我看到儿子回家后，他却又马上出门了。

ようやく泣き止んだと思ったら、また泣き出した。  
正当我以为他终于停止哭泣时，他又开始哭了。

さっきまでニコニコしていたかと思ったら、突然怒り出すし、本当に彼は気分がよく変わるなあ。  
我在想刚才他还一脸笑意时，突然他就生气了，他的心情真是变幻莫测呀。
### 笔记
表示前面的事情刚发生，紧接着就出现了<b>讲话人（第一人称）意想不到</b>的后项。前后<b>多为相反或相对立的事项。可以是一次性的事情，也可以是反复出现的情况。</b>

## 16.〜か〜ないかのうちに
### 释义
中 : 一.....就......； ......的同时。  
日 :～が終わると同時に、次のことが起こる。
### 接续
动词辞书形 + か + 动词ない形 + かのうちに  
动词た形 + か + 动词ない形 + かのうちに
<b>动词为同一个动词</b>  

### 例句
一郎はベッドに横になるかならないかのうちに、ぐっすり眠ってしまった。    
一郎一躺到床上就酣然入睡了。

わたしは夜が明けたか明けないかのうちに家を出て、空港へ向かった。    
天一亮我就从家里出发前往机场了。  
### 笔记
······结束的同时，发生下一件事情。   
<b>前接表达瞬间结束意义的动作和变化的动词（着く・終わる）等。</b>      
因为用于实际发生的事情，后面<b>不接表达说话人的意志、以及要求听话人做某事的句子。</b>   
「～（か）と思うと・～（か）と思ったら」相比，“几乎同时发生”的语气更强烈。    
在接续た形的时候ない前面的动词<b>不需要使用</b>た形   

## 17.〜かねない
### 释义
中 : 可能导致……（不好的结果）  
日 : 〜してしまうかもしれない / 〜という悪い結果になる可能性がある  
### 接续
动词连用形 + かねない  
### 例句
それは命を落としかねない感染症らしい。  
那好像是一种可能导致死亡的传染病。  

仕事のやりすぎは、鬱の原因になりかねない。  
过度劳累可能是导致抑郁的原因。  

ちゃんと毎日勉強すれば、来週のテストは合格できかねない。  
如果我每天好好学习，也许就能通过下周的小测。  

彼女はどんな病気でも治しかねない。  
说她能治愈任何疾病也不为过。  
### 笔记
〜かねる : 无法……；难以…… ,所以かねない就是 不难 有可能
因为词尾是ない所以可以当作一类形容词使用

## 18.〜かねる
### 释义
中 : 无法……；难以……  
日 : 〜できない  
### 接续
动词连用形 + かねる  
### 例句
私は責任者ではございませんので、その質問にはお答えしかねます。  
因为我不是负责人，所以我无法回答这个问题。  

あなたの気持ちも理解できますが、その案には賛成しかねます。  
虽然我理解您的心情，但我还是无法赞成那个的提案。  
### 笔记
表示从<b>说话人的立场和状况出发</b>，很难做某事，心里上有抵触感，<b>多用于服务行业。委婉地拒绝别人。</b>   
一般多用于表示不好的结果和（かねない）相似  

## 19.〜かのようだ・〜かのように
### 释义
中 : 有点……的感觉。  
日 : 〜ように見える / 〜ように感じる  
### 接续
动词简体形 + かのようだ・かのように  
形容词简体形 + かのようだ・かのように  
形容动词词干 + である + かのようだ・かのように  
名词 + かのようだ・かのように  
名词 + である + かのようだ・かのように  
### 例句
彼女は辛い出来事があったにもかかわらず、何事もなかったかのように明るく振舞っていた。  
尽管经历了痛苦不堪的过去，但她却表现得好像什么也没发生过一样，性格十分开朗。  

彼はまるで全てを知っているかのように話しているが、全て彼の想像にすぎない。  
虽然他好像什么都知道似的说着，但这一切只不过是他的想象。    
### 笔记
因为以か开头 所以接续都与か类似  
か的接续和名词类似  

## 20.〜からいうと・〜からいって・〜からいえば
### 释义
中 : 从……来看。   
日 :〜を考えると / 〜から判断すると  
### 接续
名词 + からいうと・からいって・からいえば   
### 例句
A：今度の試合、４番は誰がいいと思う？  
B：実力からいって、木村さんがいいと思います。    
A：这次比赛第四位派谁出场好？   
B：就能力而言，我觉得派木村出场好。    

模擬試験の結果からいうと、ジョンさんはN1に合格できるだろう。  
从模拟考试的成绩来看，约翰N1考试应该能通过。    
### 笔记 
表示以某一立场、某一侧面为评论的依据，对事物进行是“是”还是“非”、是“好”还是“坏”、是“对”还是“错”、是“行”还是“不行”等评价或判断。
