# N2语法学习笔记Day4

## 31.〜ことなく
### 释义
中 : 没有……，不……。  
日 : 〜ないで   
### 接续
动词辞书形 + ことなく   
### 例句
プロジェクトの進捗が遅れているので、休日も休むことなく働いている。  
因为项目的进度太慢了，所以休息日也没休息，还在继续工作。  

彼は最後まであきらめることなく、頑張った。  
他直到最后都没有放弃，好好努力了。  

この時計は１０年止まることなく、動き続けている。  
这座钟已经运转了10年，从未停止过。    
### 笔记
〜ことなく 是一种较为正式和书面的表达，因此在日常生活中如果用它来描述过于琐碎或普通的事情，可能会显得不自然。  
“〜ことなく” <b>更适用于描述重要、正式或有一定意义的行为，例如努力、决心或某种值得强调的情况。</b>   
<b>不适用于琐碎的日常行为（如吃饭、买东西等）。</b>   

## 32.〜ことだろう・〜ことか
### 释义
中 : 感到非常……。  
日 : 非常に〜だと感じる。  
### 接续
动词简体形 + ことだろう・ことか  
形容词简体形 + ことだろう・ことか  
形容动词词干 + な + ことだろう・ことか  
名词 + である + ことだろう・ことか   
### 例句
君に会えてどんなに嬉しかったことか。    
能和你见面我会多么开心呀。    

この参考書を何年使ってきたことだろう。本当に役に立った。    
这个参考书用了好几年了，真的太实用了。   

会社の先輩との飲み会は楽しくないのに、友達との飲み会はなんて楽しいことだろう。    
明明和公司的前辈的酒会不开心，但和朋友的酒会就非常开心。    
### 笔记
「〜ことだろう」和「〜ことか」 是表示感叹的表达方式，用来描述说话人强烈的感情（如喜悦、悲伤、感动、惊讶等），可以翻译为：  
「多么……啊！」  
「该有多……啊！」  
它们可以用于各种感情，但在语气上带有一定的强调色彩。  
这些表达常与一些加强感叹的词结合使用，如：  
「なんと」（多么）  
「なんて」（多么）  
「どれほど」（多么、何等）  
「どんなに」（多么）

## 33.〜ことになると・〜こととなると
### 释义
中 : 一聊到……，就……。    
一牵扯到……，就……。    
日 : ～の話題になると / ～の話になると  
### 接续
动词辞书形 + ことになると・こととなると  
名词 + の + ことになると・こととなると   
### 例句
田中さんはアクション映画のこととなると、話が止まらなくなる。  
田中只要一聊到关于动作电影的话题，就会变得滔滔不绝。  

部長はいつも怖いが愛猫のことになると、優しくなる。  
经理虽然一直很可怕，但当谈到他心爱的猫咪时，就会变得温柔。  
### 笔记
と在N3的语法中本身就带有 一......，就.......。的含义  
这个表达的主要功能是强调在特定的场合或话题下，与平常不同的变化或反应。  
它往往带有一种对变化的关注，常用于描述态度、行为、情绪、情况的改变。 

## 34.〜ことに（は）
### 释义
中 : 非常……；十分……    
日 : とても～で  
### 接续
动词た形 + ことに（は）    
形容词辞书形 + ことに（は）    
形容动词词干 + な + ことに（は）     
### 例句
悔しいことに、1点差で試合に負けてしまった。    
太不甘心了，因为一分之差输了比赛。      
### 笔记
「〜ことに（は）」 用于表达说话人对某件事的强烈情感，并强调这种情感。    
翻译为： “令人……的是” “让人……的是”    
「〜ことに（は）」后面不能接意志或命令等表示主观意图的句子，因为这个表达强调的是已经发生的事实或客观状态。    
「〜ことに（は）」 和某些表示情感的词语常常搭配使用。这些词汇用于表达说话人的具体感情：    
正面情感（积极情绪）：嬉しい（开心的）ありがたい（感激的）面白い（有趣的）  
负面情感（消极情绪）：悲しい（悲伤的）悔しい（懊悔的）残念な（遗憾的）困った（让人为难的）驚いた（吃惊的）  

## 35.〜ざるを得ない
### 释义
中 : 不得不……；只好……  
日 : ～しなければならない / どうしても〜する必要がある
### 接续
动词ない形去ない + ざるを得ない  
※する → せざるをえない      
### 例句
足が痛く、歩ける状況ではないので、病院へ行かざるを得ない。  
因为脚痛得走不了路了，所以只好去医院。  

工場の排気を低くせざるを得ない。  
我们只好降低工厂的废气排放。  
### 笔记
・属于书面语，表示除此别无他法。  
・包含 “本来不想做，但是迫于某种压力或原因而不得不去做” 的意思。  
<b>特别注意记忆 する 的接续 !!!!</b>

## 36.次第（しだい）
### 释义
中 : ……完成之后，马上……  
日 : 〜したらすぐに / 〜が終わったらすぐに  
### 接续
动词连用形 + 次第   
### 例句
会場の準備ができ次第、ご案内いたします。もうしばらくお待ち下さい。  
会场的准备工作一结束，我们会马上带大家过去。请再稍等片刻。  

詳しいことがわかり次第、ご連絡いたします。  
事情一有进展，我会马上通知您。  
### 笔记
・书面语。  
・……实现以后，马上采取后续行为。  
・<b>前接表示“时间到了自然会发生”的事件实现瞬间的词语。后接表达说话人的希望、意志，以及要求听话人做某事的句子。</b>  

## 37.〜次第だ・〜次第で
### 释义
中 : 由……决定  
日 : 〜によって決まる
### 接续
名词 + 次第だ・次第で  
### 例句
日曜のイベントに参加するかどうかは天気次第かな。  
参不参加星期天的活动要根据天气的情况来决定。  

同じ化粧品でも、使い方次第では効果が変わってくることがあります。  
即使是同一个化妆品，根据使用方法的不同，效果也会有所不同。  

台風の影響で交通機関の運行が乱れたため、商品のお届けが遅れた次第です。  
由于台风导致运输中断，商品的交付被推迟了。  
### 笔记
表示后项会产生怎样的结果，全凭前项条件如何。后项会不会发生什么变化，要取决于前项的具体情况。  
#### 〜次第だ
表示某事的结果、原因或经过，即说明理由或解释过程。  
类似于中文的**“因此……”、“就是因为……”** 或 “取决于……”。  
多用于总结说话人对某件事的态度或决定。  
#### 〜次第で
表示某件事根据某种条件而变化，或强调取决于某事。  
类似于中文的**“取决于……”** 或 “根据……”。  

## 38.〜末(すえ)（に）
### 释义
中 : 经过……，最后……。  
日 : 色々〜したけど、最終的に。
### 接续
动词た形 ＋ 末（に）  
サ变名词 ＋ の ＋ 末（に）  
### 例句
数回におよぶ議論の末、Aの案を採用することにした。     
多次讨论之后，最终决定采用A方案。   

色々悩んだ末、もうしばらくこの会社で頑張ってみることにしました。   
纠结了很久，最后还是决定在这个公司试试。      
### 笔记  
表示结果。    
包含“经过一番……最后终于……”的意思，强调经过了一个很长的过程，最后得出某个结论或者产生某个结果。       
类似于中文的 “经过……之后，最终……”，强调经过了长时间或复杂的过程。　　

## 39.〜ずじまい
### 释义
中 : 没……就结束了。    
日 : 〜しないで終わる。  
### 接续
动词ない形去ない + ずじまい  
※しない → せずじまい  　
### 例句
ジョンさんとはケンカしてから、仲直りできずじまいで卒業してしまった。 
和约翰吵架之后，还没跟他和好就毕业了。  

マウンテンバイクを買ったが、使わずじまいで売ってしまった。  
虽然买了一辆山地自行车，但没有使用就卖掉了。   
### 笔记  
不了了之，带有后悔，遗憾的口气。      
注意 动词ず形接续    
しまい本身就有结束的意思 也带有遗憾等消极的情感  
「ず」+「しまい」→「ずじまい」  
「しまい」中的 「し」 和前面的 「ず」 发生连音，简化为 「じ」。  
这种现象属于日语中常见的「サ行音变」，让语感更加自然、顺口。  

## 40.〜ずにはいられない・〜ないではいられない
### 释义
中 :忍不住……；不得不……。  
日 : どうしても〜しないでいることはできない。/ どうしても〜してしまう。
### 接续
动词ない形去ない + ずにはいられない・ないではいられない  　　
※しない→せずにはいられない  　　
### 例句
この店のカレーはとても辛い。ちょっと食べたら、水を飲まずにはいられない。  
这家店的咖喱太辣了，稍微尝一点就忍不住猛喝水。  

この映画を見ると誰でも泣かずにはいられない。  
看了这部电影的人不论是谁都会忍不住落泪。   
### 笔记  
表示无法抑制某种情感或行为的表达，带有一种“不由得”的语气。  
本身在N3里面就存在 ないで=ずに 的说法 因此两种表达几乎等价 但是 ずに 因为较为古代一点 因此更加正式  
いられない就是 いる 的可能性否定 所以意思是 不可能不做某事 也就是不由得做某事  

## 总结
31~40的语法里面和こと相关的居多 注意こと的前后接续即可  
39 40两个语法要求记忆动词的ず形变形以及含义
