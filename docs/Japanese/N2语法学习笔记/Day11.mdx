# N2语法学习笔记Day11

## 101.〜ことだから
### 释义 
中 : 正因为是……  
日 : 〜なのだからきっと  
### 接续
动词简体形 + ことだから  
形容动词词干 + な + ことだから  
名词 + の + ことだから  
### 例句
トムさんのことだから、余裕でN1にも合格するでしょう。  
正因为是汤姆，N1应该很轻松就能合格吧。  

いつも遅れてくる田中くんのことだから、今日も遅れてくるだろう。  
正是因为是一直都迟到的田中，所以今天也会迟到吧。  

お父さんとお母さんが珍しくうちに来ていることだから、久しぶりにみんなで映画でも見よう。  
正因为爸妈难得来我家，所以我们久违地一起去看场电影什么的吧。  
### 笔记
#### 「〜ことだから」
根据某事的特点或背景，推测或解释一个结果  
主观、推测性，语气较温和，适合描述熟悉的人或事  
#### 「だから」
明确表示原因和结果之间的逻辑关系  
客观、直接，强调因果关系  

## 102.〜のもとで
### 释义 
中 : 在……的影响下做某事，处于……的状态下做某事。  
日 : ～の影響を受けてあることをする・～の状況であることをする。  
### 接续
名词 + のもとで・のもとに  
### 例句
わたしは生まれてからずっと、優しい祖母のもとで幸せに暮らしてきました。  
我自打出生以后，一直在慈祥的祖母身边幸福地生活着。  

この研究所では、一定の温度と湿度のもとで育てられた植物から新しい薬品を作り出した。  
这个研究所从在特定的温度和湿度条件下培育出来的植物中研发出了一种新药品。  

校庭でのキャンプファイヤーは夏の最大イベントで、周辺住民の了解のもとに10年も続いている。  
校园篝火晚会是夏季最大的文娱活动，在得到居民理解的情况下，已经持续十多年了。  
### 笔记
#### 「～のもとで」
主要前接表示人的词语（先生・両親）等，含有“受到那个人影响”之意。  
后面多接表示行为的句子（例：励む・暮らす）。  
#### 「～のもとに」
主要前接表示状况含义的名词（例：管理・了解），含有“在此条件、状况下”的意思。  
后续表示行为、状态的句子（例：続いている・平等だ）。  

## 103.〜ばかりか
### 释义 
中 : 不仅......而且...... ,    
连......也......    
日 : ～だけでなく、さらに。    
### 接续
动词简体形 + ばかりか    
形容词辞书形 + ばかりか    
形容动词词干 + な/である + ばかりか  
名词 + である + ばかりか    
### 例句
Aコースの山道は、初心者には危険なばかりか、途中の景色もあまり良くない。  
路线A的山路，不仅对于初学者而言很危险，而且沿途的景色也不怎么好。  

発見が遅れたばかりか対策にも手間取ったので、牛の病気が広がってしまった。  
不仅发现晚了，而且订制对策也花费了时间，所以牛的疫情就蔓延了。  
### 笔记
前句叙述不寻常的事情，后句再次追加使人感到意外的其他事情。  
后句多使用表达“其他也”含义的「も」。后面不接要求听话人做某事的句子。  

## 104.〜ばかりだ
### 释义 
中 : 变化朝着......单一方向不断加剧。  
日 : ～という一方方向に変化が進んでいく。  
### 接续
动词辞书形 + ばかりだ  
### 例句
このごろは仕事が多くて残業が増えるばかりだ。  
最近工作很多，一直在不停地加班。  

東京の交通機関は複雑になるばかりで、わたしはよくわからなくなってきた。  
东京的交通越来越复杂，我越来越搞不懂了。  
### 笔记
前接表示变化的动词（「増える・悪くなる」等）。  
「～ばかりだ」多表示朝着不好的方向发展。  

## 105.〜ばかりに
### 释义 
中 : 仅仅因为……  
日 : ～ので / ～が原因で  
### 接续
动词た形 + ばかりに  
动词ない形 + ばかりに  
形容词辞书形 + ばかりに  
形容动词词干 + な/である/だった + ばかりに  
名词 + な/である/だった + ばかりに  
### 例句
ボールをキャッチしたばかりに、肩が外れました。  
仅仅因为接住了一个球，我的肩膀就脱臼了。  

釣りに行きたいばかりに、仕事を休んで海に行った。  
仅仅因为想去钓鱼，就请了假去海边。   

おとなしそうに見えた犬の頭を撫でたばかりに噛まれたんだ。まだ手が痛いよ。   
就因为摸了一只看起来很温顺的狗狗的头，我就被咬了。手到现在还是很痛。   
### 笔记
就因为前项的原因，才导致了后项预想不到的坏结果。暗示一种“如果不出现前项就好了”、“要是不做前项就好了”等后悔、遗憾的心情。  
接在表示愿望的【たい/ほしい】后面时表示抱有这样的愿望，才去做了并不愿意做的事情。  
这个句型多数用在责怪自己（或属于自己一方的人）的行为上或抱怨自己不具备某种条件。  

## 106.〜はさておき
### 释义 
中 : 由于还有别的更为重要的事情，所以……姑且不论。  
日 : ほかにもっと大切なことがあるので、～はとりあえず話題からはずす。  
### 接续
名詞 +（助詞）+ はさておき  
### 例句
飲み会をするなら、細かいことはさておき、まずは場所と時間を決めなくては。  
要办酒会的话，细节姑且不论，应该首先确定地点和时间。  

どんな家がいいかはさておき、どんな地域に引っ越したいかを考えよう。  
姑且不论什么样的房子好，先考虑想搬到什么样的地方吧。  
### 笔记
前面多接表示在此之前已成为话题的词语。也可以前接“疑问词～か”。  
后续与前项相比更优先的、更基本的事情。  

## 107.〜はともかく（として）
### 释义 
中 : 暂且不谈；先不说  
因为有其他更想强调的事情，所以……暂不考虑。  
日 : ほかにもっと強調したいことがあるので、～はとりあえず考えないでおく。  
### 接续
名词 +（助词）＋ はともかく（として）  
动词简体形 ＋ の/かどうか ＋ はともかく（として）  
形容词简体形 ＋ の/かどうか ＋ はともかく（として）  
形容动词词干 ＋ の/かどうか ＋ はともかく（として）  
### 例句
外ではともかく、家の中でたばこを吸うのはやめて。  
在外面如何姑且不论，请不要在家里吸烟。  

受験するかどうかはともかく、願書だけはもらっておこう。  
先不管是否参加考试，至少先领取申请书吧。  

アラビア語を習いたい。読むのはともかくとして、簡単な会話はできるようになりたい。  
我想学阿拉伯语。阅读姑且不论，希望能说简单的对话。  
### 笔记
前接表达说话人认为与后项相比没有那么重要的事情的词语。  
也可以前接“疑问词～か”。后续与前项相比更想强调的事情。  

## 108.〜ならまだしも
### 释义 
中 : 如果只有……还行，但……  
日 : 〜なら少しはいいが / 〜はまだ許容範囲であるが  
### 接续
动词简体形＋ならまだしも  
名词 + ならまだしも・はまだしも  
形容词简体形＋ならまだしも  
形容动词词干＋ならまだしも  
### 例句
寿命で死ぬならともかく、殺されるのだけは嫌だ。  
宁愿死于自然寿命终结，也不愿被杀害。  

可愛いいならまだしも、可愛くもないのにわがままだ。  
如果可爱还勉强可以接受，但不可爱还任性。  

一度や二度ならまだしも、君は何度も同じ間違えをしすぎだぞ！  
只有一两次就不说了，你已经犯过很多次同样的错误了！  
### 笔记
・注意辨析「～ならまだしも」与「～はまだしも」  
・「～ならまだしも」表“……的话还算说得过去，可竟然……”“……的话还能原谅，可竟然……”之意  
・「～はまだしも」表“虽然还不能说足够令人满意，不过也算差强人意了”之类的意思   

## 109.〜はもとより
### 释义 
中 : ……自不必说，其他也如此。  
日 : はもちろん、ほかもそうだ。  
### 接续
名詞 +（助詞）+ はもとより  
### 例句
地元の住民はもとより、周辺の地域に住む人たちも原子力発電所に不安を感じている。  
当地居民自不用说，居住在周边地区的人们也对核电站感到不安。  

この学校では、教室内ではもとより教室の外でも禁煙を守ってください。  
在本校，教室内自不用说了，教室外也严禁吸烟。  
### 笔记
前接表达说话人认为理所当然的例子的词语。  
后续其他的例子。  
后面多接表达“其他也”含义的「も」的句子。  
书面语。  

## 110.〜反面
### 释义 
中 : 表示两面性或对比关系。   
日 : ～だが、逆に…の面もある。     
### 接续
名词 + である ＋ 反面   
形容动词词干 + である ＋ 反面   
形容动词词干 ＋ な ／である ＋ 反面  
动词辞书形 ＋ 反面   
その＋反面（では）  
### 例句
駅に近い反面、騒音問題に悩まされている。  
离车站很近，但另一方面却被噪音问题所困扰。  
### 笔记
#### 表示对比
某事物具有两个相对或矛盾的方面。  
常用于描述优缺点或两个不同的性质。  
#### 表示两面性或并存关系  
同一事物中存在两种特性，虽然不一定矛盾，但会形成对比。
