# N2语法学习笔记Day3

## 21.からして
### 释义
中 : 单从……来看。  
日 : 〜から判断して。 
### 接续
名词 + からして  
### 例句
この本はタイトルからしてつまらなさそうだ。  
这本书看标题就觉得很无聊。  

この家は玄関からして立派だ。きっと、お金持ちが住んでいるんだろう。  
这座房子从入口处看起来就很漂亮。一定住着有钱人吧。  
### 笔记
表示判断的依据。一般用于负面情况。

## 22.〜からすると・〜からすれば
### 释义
中 : 站在……的立场考虑的话  
日 : 〜の立場で考えると  
### 接续
名词 + からすると・からすれば  
### 例句
ネイティブの日本人からすると、トムさんの日本語はまだまだです。  
在日语母语者的眼中，汤姆的日语水平还不够格。  

お金の持ちのトムさんを羨む人が多いですが、私からすると最低限生活できるお金があれば幸せなので、何とも思いません。  
很多人羡慕有钱人汤姆，但我看来，只要有能够过活的钱我就很高兴了，所以我不认为这有什么。  
### 笔记
表示以某一立场、某一侧面为观察的依据，对事物进行“是”还是“非”、是“好”还是“坏”、是“对”还是“错”、是“行”还是“不行”等评价或判断。  

## 23.〜からといって  
### 释义
中 : 虽然……。    
日 : たとえ〜という理由があっても。  
### 接续
动词简体形 + からといって  
形容词简体形 + からといって  
形容动词词干 + だ + からといって  
名词 + だ + からといって  
### 例句
頑張ったからといって全ての人が必ず報われるわけではない。  
虽说努力了，但并不是所有人都会得到回报。  

自分が好きだからって、相手にそれを押し付けるべきではない。  
即使自己喜欢，也不应该强加给对方。  
### 笔记
如果仅是……这个理由的话，那么由此理由引发的一般性的推测不能成立。  
这是用来否定“因为……所以理所当然就……”的表达形式。后面多接部分否定的表达方式「～とは限らない・～わけではない・～とはいえない」等。    
换成中文的话也就是<b>即使......,也.......。</b>  

## 24.気味   
### 释义
中 : 稍微，有点……的感觉  
日 : いつもと違って、少し〜の感じがする       
### 接续
动词连用形 ＋ 気味  
名词 ＋ 気味  
### 例句
物価は上がり気味だ  
物价有点上涨的趋势。  

このところ忙しくて、疲れ気味だから、今日は早く帰ることにした。  
因为最近忙得感觉有点累了，所以决定今天早些回去。  
### 笔记
表示带有某样子或某倾向的意思，<b>一般用于不好的情况</b>  

## 25.〜きり   
### 释义
中 : 做了……之后，就再也没有……。   
日 : "〜してから、そのままずっと"  
### 接续
动词た形 + きり  
### 例句
息子は今日、出かけたきり、夜になっても帰ってこないのでとても心配です。  
儿子今天出门后，到现在都那么晚了还没回来，我很担心他。  

お祖母ちゃんは去年病気で倒れて、寝たきりになった。  
奶奶去年因病倒下，变得卧床不起了。  

父がついに寝たきりになってしまった。  
我父亲已经完全卧床不起了。  

彼女とは喧嘩したっきり会っていません。  
自从和她吵架之后我就没再见过她了。  
### 笔记
有两种使用方法    
第一种是　Vたきり、......ない　表示做完前者之后再没有做过后者了   
第二种是　Vたきり(になる) 表示在此之后一直保持前者的状态没有改变　  　
### 释义
中 : 只……，仅仅……。  
日 :〜だけ  
### 接续
名词 + きり　　  
### 例句
今度、二人きりで食事しない？　　
下次，就我们两个人一起吃个饭吧。　　
一度きりの人生をどう生きるかは個人の自由だ。　　
如何度过仅此一次的人生是个人的自由。　　
### 笔记
与だけ的意思类似 　
也可以接续在それ　これ　あれ等等的后面  
无论哪种意思的きり　口语都有　っきり、キリ 两种形式  

## 26.〜くせに・〜くせして   
### 释义
中 : 明明……，却……  
日 : 〜という立場・身分なのに
### 接续
动词辞书形 + くせに・くせして  
形容动词词干 + な + くせに・くせして  
名词 + の + くせに・くせして  
### 例句
のび太のくせに生意気だぞ！  
区区大雄，真是太嚣张了！  

斎藤さんは日本人のくせに、日本語が変なんだよな。  
斋藤虽然是日本人，但他说的日语很奇怪。  
### 笔记
くせ　:　癖 因此前面接续跟名词类似  
多用于负面的不好的情况  
表示前项的事实根本不符合后项的内容。用于讲话人指责、鄙视他人的言行举动。　　

## 27.〜くらいなら・〜ぐらいなら  
### 释义
中 : 比起……，不如……。  
日 : 〜よりも、むしろ・・・のほうがいい。
### 接续
动词辞书形 + くらいなら・ぐらいなら  
### 例句
のび太のくせに生意気だぞ！    
区区大雄，真是太嚣张了！    

斎藤さんは日本人のくせに、日本語が変なんだよな。    
斋藤虽然是日本人，但他说的日语很奇怪。   
### 笔记
「Aくらいなら、むしろB」是先提出最差的情况A，彻底否定A。然后选择与其对立的B。    
中文意思与 与其...，不如...类似    
所以常跟 寧ろ（宁可 不如） 搭配　　

## 28.〜げ  
### 释义
中 : 好像……；带有……的样子  
日 : 〜そうだ / 〜そうな様子だ  
### 接续
形容词词干 + げ  
形容动词词干 + げ  

动词连用形 + げ  
名词 + げ  
名詞や動詞に接続することもあるが、使えるものは限定的。  
### 例句
彼の表情は少し、悲しげだった。  
他的表情似乎有些悲伤。    

あの女の人は寂しげにワインを飲んでいる。  
那个女人一脸寂寞地喝着葡萄酒。  
### 笔记
构成新的形容词，表示由外部的观察而感觉到的情况、样子、感觉等。    
意思跟「そうだ」相似，一般只用于书面上    
「〜げ」：通过外表或行为猜测情绪、氛围。（主観的）　　  
「〜らしい」：更关注事物本质或典型特征是否符合。（客観的）　　  
接续构成的新形容词类似于二类形容词的词性    

## 29.〜ことだ  
### 释义
中 : 应该……。  
日 : 〜しなさい / 〜したほうがいい    
     〜するな / 〜しないほうがいい  
### 接续
动词辞书形 + ことだ  
动词ない形 + ことだ   
### 例句
日本語が上手になりたかったら、毎日話すことだ。  
想要日语能力变强的话，需要每天都说日语。  

悔しかったら、もっと強くなることだね。  
不甘心的话，就应该变得更强。  
### 笔记
表示忠告或建议等，用于说话人对对方的忠告、提醒等，建议对方这样做会更好。  
#### 「ことだ」：  
多用于正式场合或较严肃的对话，适合上对下或平辈之间的劝告。  
##### 例：  
試験に合格したければ、もっと努力することだ。  
（如果想通过考试，就必须更加努力。）  
#### 「べき」：  
强调个人或社会的普遍价值观，适合表达自己强烈的信念，也可以用于对他人的较强劝告。  
##### 例：  
学生は勉強すべきだ。  
（学生就应该学习。）  
#### 「ほうがいい」：  
更适合日常交流，语气友好，用于提建议或比较两种选择。  
##### 例：  
こんな時間に行くより、明日行ったほうがいいよ。  
（与其这么晚去，不如明天去更好。）  

## 30.〜ことだし     
### 释义  
中 : 正因为……；也……。    
日 : ので / から    
### 接续
动词简体形 + ことだし    
形容动词词干 + な + ことだし    
形容词简体形 + ことだし    
名词 + の + ことだし    
### 例句
せっかく海外にいることだし、日本食じゃないものを食べようよ。    
难得也出国了，去吃点不是日本菜的东西吧。    

子どもが風邪をひいていることだし、今晩の飲み会は参加しないつもりです。    
因为我的孩子感冒了，所以今晚的酒会我打算不参加。    
### 笔记
軽い理由を表す言い方。    
表理由的还有<b>から　ので　ために</b>等。  

## 总结
第21~30个语法中大部分和か　から有关  
接续可以着重记忆か　から的然后进行扩展  
### から
动词简体形 + から    
形容词简体形 + から    
形容动词词干 + だ + から    
名词 + だ + から    
在其他跟から有关的文法里面 接续都类似
