# N2语法学习笔记Day7

## 61.〜ところだった
### 释义
中 : 再……一点，可能就会……    
日 : もう少しで～になる可能性があった。/ ～という結果になったかもしれない。  
### 接续
动词辞书形 + ところだった  
动词ない形 + ところだった  
### 例句
この電車に間に合ってよかった。もう少しで待ち合わせ時間に間に合わないところだった。  
能赶上这趟电车真是太好了。差点就错过集合时间了。  

現金が足りなくて家賃が払えないところだった。  
现金不够，快要付不起房租了。  
### 笔记
「〜ところだった」 用来表示某个事件或动作本来可能发生，<b>但最终并未发生</b>。  
<b>它通常搭配动词的 辞书形（基本形） 来表示。</b>  

## 62.〜どころではない・〜どころじゃない
### 释义
中 : 不是……的时候；哪有(时间)……  
日 : ～はとてもできない / ～する時間がない
### 接续
动词辞书形 + どころではない・どころじゃない  
动词ている形 + どころではない・どころじゃない  
形容词辞书形 + どころではない・どころじゃない  
名词 + どころではない・どころじゃない  
### 例句
アメリカに住み始めた頃は、英語の勉強に必死で、遊びどころではなかった。    
刚开始在美国居住时，就拼命学习英语，哪有时间玩耍。  

休みの日はたくさん寝たい。レジャーどころではない。  
休息日我想多补充睡眠。哪是去参加休闲活动的时候。  

今まで恋人も出来たことがないのに、結婚どころではない。  
明明还从未有过对象，谈什么结婚。  
### 笔记
表示不是做某事的场合，强调目前没有时间，金钱和精力去做某事。  

## 63.〜として〜ない
### 释义
中 : 完全不……  
日 : 全く〜ない  
### 接续
1 + 量词 + として〜ない  
### 例句
君のことを１日として考えない日はないよ。本当に愛してる。  
我没有一天不想你。我真的很爱你。  

津波により家が流されてしまい、何１つとして残っていない。  
房子由于海啸被冲毁了，什么也没有留下。  
### 笔记
在“一”的助数词后面接否定，表示全盘否定。  

## 64.〜とともに
### 释义
中 : 随着……，（与之相应）渐渐变得……  
日 : ～すると、それに応じてだんだん/～が変化すると、・・・も一緒に変化する  
### 接续
动词辞书形 + とともに  
名词 + とともに  
### 例句
年を取ると共に、身体がどんどん硬くなってきた。  
随着年龄的增长，身体变得越来越僵硬。  

春が近づくとともに、少しずつ暖かくなってきた。  
随着春日渐近，天气逐渐变得温暖起来。  
### 笔记
「〜とともに」 可以用来表示“与…一起”或“和…一起”。这时，表示的是伴随、一起做某事的关系。  
「〜とともに」 还可以表示“随着（某事物的变化）”，通常用来表达某个情况或状态随着另一个事物的发展或变化而变化。  
作为“与…相伴随”的意思 这个用法表示两件事物或现象一起发生，互相伴随。它常常用于描述两个事物或现象之间的关联。  

## 65.〜となると・〜となれば・〜となったら
### 释义
中 : 如果……的话  
日 : もしそうなったら
### 接续
动词简体形 + となると・となれば・となったら  
形容词简体形 + となると・となれば・となったら  
形容动词词干 + となると・となれば・となったら  
名词 + となると・となれば・となったら
### 例句
大型台風が上陸するとなると、周辺の農作物にかなり影響が出るだろう。  
如果强台风登陆，将对周边地区的农作物造成重大影响。  

ゴールデンウィークともなると、遊園地などは混む。  
一到黄金周，游乐园之类的地方就会变得非常拥挤。  

ここは普通の漁村だが、夏休みともなると大勢の客が来て、賑やかになります。  
这里是普通的渔村，但是一到暑假，大量的游客涌来，就会变得热闹起来。  

大人ともなると、幼馴染との友情は弱くなりつつあるだろう。  
一旦变成大人后，与儿时好友的联系就会变得越来越淡吧。  
### 笔记
・在想要表达“如果发生了那样的事的话，会有别的新的事发生”之意时使用本句型。　  
・表示“如果、假如”，描述的是假设一件（尚未发生的）事发生，表示“如果已经决定为那样”，描述的是既定的（已经发生的）事。  

## 66.〜とは
### 释义
中 : 对……这个词语进行说明，下定义。  
日 : ～という言葉を説明・定義する。
### 接续
名詞 + とは  
### 例句
「増悪」とは医学用語で、症状がもっと悪くなることである。  
“增恶”是医学用语，是指症状恶化的意思。  

人生とは本人が主役のドラマみたいなものだ。  
所谓人生就好像是一部由自己主演的戏。  
### 笔记
是「～というのは」的更书面的表达方式。  
后续对该词的意思或本质进行说明的句子「～だ・～である・～という意味である・～（の）ことである」等。  
书面语。


## 67.〜に越したことはない
### 释义
中 : 没有什么比……更好；……更安全  
日 : 常識的に考えて〜の方がいい。〜の方が安全だ  
### 接续
动词简体形 + に越したことはない  
形容词简体形 + に越したことはない  
形容动词词干 + である + に越したことはない  
名词 + である + に越したことはない  
### 例句
応援してもらうことに越したことはないが、ちょっとやり過ぎです。  
没有什么比有人为你加油更好的了，但我认为他们做得太过了。  

家は広いに越した事はないが、値段と場所も大切だ。  
没有什么比宽敞的房子更好的了，但价格和位置也很重要。 

億万長者であるに越した事はない。  
没有什么比成为亿万富翁更好的了。  
### 笔记
「～に越したことはない」<b>通常呈现一些积极的东西，或者一些希望得到的理想结果</b>。  

## 68.〜にこたえて
### 释义
中 : 为了迎合……；为了回应……。  
日 : 〜を受け入れて 、 〜合うように  
### 接续
名词 + にこたえ（て）  
名词 + にこたえる + 名词  
### 例句
あなたの要望に応える自信はありますが、このデザインを完成させるには時間がかかると思います。  
我有信心能够回应您的需求，但完成此设计需要时间。  

クライアントの要望にお応えて、新しい機能を追加いたしました。  
为了响应客户的需求，我们追加了新功能。  
### 笔记
顺应各种要求、期待、爱好等，做出相应的回应。<b>后项的行为都是为了实现前项而做的。</b>      

## 69.〜に際して
### 释义
中 : 当……之际；当……的时候  
日 : 〜するときに / 〜に先立ち  
### 接续
动词辞书形 + に際して  
サ变名词 + に際して  
サ变名词 + に際して + の + 名词  
### 例句
彼は勉強が嫌いだった。授業中はあくびばかりしていた。しかし卒業に際して先生との別れを一番悲しんだのは彼だった。  
他讨厌学习，上课期间总是在打哈欠。可是当毕业之际，要跟老师道别时，最伤心的却是他。  

ローレンス様とのご契約に際しまして、当方の契約書に誤りがございましたことをお詫び申し上げます。  
劳伦斯先生，在与您签订合同时，我们这边的协议存在错误，对此我深表歉意。  
### 笔记
「～に際して」主要用于在……之前要做的事情时，或者当出现……的情况时  

## 70.〜に限って
### 释义
中 : 仅在……的情况下，绝对不……；  
仅在……的情况下，特别会……。  
日 : 〜の場合だけ、絶対に〜ない。  
〜の場合だけ、特にそうである / 運悪くそうなる。  
### 接续
名词 + に限って  
### 例句
急いでいる時に限って、何でいつも渋滞なのよ。  
为什么赶时间的时候，总是堵车呢。  
### 笔记
后项接表示“与以往不同，很特别”的句子。
