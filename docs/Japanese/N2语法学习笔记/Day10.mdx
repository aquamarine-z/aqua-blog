# N2语法学习笔记Day10

## 91.〜につき
### 释义 
中 : 因为……；由于……  
日 : 〜のため、〜という理由で
### 接续
名词 + につき  
### 例句
店内改装中につき、しばらく休業いたします。  
店铺翻新，暂时休业。  

閉店につき、特別セール開催中！  
本店关门，特价促销中！  
### 笔记
・表示原因，是很郑重的表示原因的用法  
・常用于通知、告示、书信等  
表示原因的还有「のために」「のように」「によって」等  

## 92.〜につけて
### 释义 
中 : 每当/一有机会……心情就总是必然如此。  
日 : ～すると、いつも必ずそういう気持ちになる。  
### 接续
动词辞书形 + につけて  
### 例句
彼女のうわさを聞くにつけて、心配になる。  
每次听到有关她的传闻就会很担心。  

父はなにかにつけて、若いころ外国で過ごした思い出を語る。  
父亲一有机会就讲述他年轻时在外国生活的经历。  
### 笔记
后续表示心理活动的句子。  
<b>「何かにつけて」是惯用表达方式，表示“每当发生什么事的时候”的意思。</b>      
<b>这种情况下，后面可以不接表示心理活动的句子。</b>      

## 93.〜に伴（ともな）って
### 释义 
中 : 伴随着……  
日 : Aが変化するとBも一緒に変化する  
### 接续
动词辞书形 + （の） + に伴って  
动词辞书形 + （の） + に伴う + 名词  
名词 + に伴って  
名词 + に伴う + 名词  
### 例句
ライト兄弟による初めての飛行は１９０３年に成された。それに伴い、近代の航空産業が生まれた。  
1903年莱特兄弟成功成为世界上第一个飞行者，现代航空业由此诞生。  

高速道路の情報サイト：リフレッシュ工事に伴う車線規制のお知らせ。    
高速公路信息网站：因改造工程而导致车道限制的通知。   

時代の変化にともなって、人々の考え方も変わってきた。   
随着时代的变迁，人们的思维方式也发生了变化。  
### 笔记
<b>伴う:ともなう</b>  
可用于同时逐渐变化的事件，也可用于突然变化的瞬时事件    

## 94.〜にほかならない
### 释义 
中 : 正是……；不外乎……  
日 : 〜以外のものではない / 絶対に〜だ  
### 接续
名词 + にほかならない  
### 例句
緊急時に一番大切なことは、まず落ち着くことにほかならない。  
遇到紧急情况时，最重要的是要先保持冷静。  

かすみさんがみんなに好かれるのはお人よしだからに他ならない。  
霞之所以受到大家的喜欢，正因为她是一个心地善良的人。  

こんな夜遅くに訪問してくるなんて迷惑に他ならない。  
这么晚来我家什么的，真是给我添麻烦。  
### 笔记
・强调 “除此之外，没有其他”，相当于「～だ」  
・在断言「绝对是/绝对不是……以外的任何事物」时使用，通常出现在评论性的文章中，为书面语   

## 95.〜にも関わらず
### 释义 
中 : 明明……；虽然……，但是……  
日 : ～のに  
### 接续
动词简体形 + にもかかわらず  
形容词简体形 + にもかかわらず  
形容动词词干 + である + にもかかわらず  
名词 + （である） + にもかかわらず  
### 例句
イベントは雨にも関わらず、多くの人が集まった。  
尽管下着雨，还是有很多人来到活动现场。  

浜崎さんは毎日一生懸命働いていたのにもかかわらず、会社をクビになった。  
尽管滨崎每天都拼命工作，他还是被公司解雇了。  

彼が悪いのにもかかわらず、私に謝らそうとしてきた。  
尽管是他不好，他还是想让我道歉。  
### 笔记
・采用「∼にもかかわらず、…」的形式，意为“与从～的事态预想到的相反，结果变成了「…」，「…」处多使用表达说话人吃惊、意外、不满、责难等语气的词句   
・<b>后项的结果与根据前项的事实做出的推测不同</b>      

## 96.〜に基づいて
### 释义 
中 : 以……为基础；把……作为根据。  
日 : ～を基礎にして / ～を根拠に。  
### 接续
名词 + に基づいて  
### 例句
警察は、男の子の証言に基づいて犯人を特定した。  
警察根据男子的证言锁定了犯人。  
### 笔记
把前项当作后项时的标准行事。  

## 97.〜ぬ
### 释义 
中 : 不……  
日 : ～ない  
### 接续
动词ない形去ない + ぬ  
动词ない形去ない + ぬ + 名词  
### 例句
私の知らぬ間に、夫が仕事をやめていた。  
在我不知情的情况下，丈夫辞去了工作。  

なんだこの飯は！こんなまずそうなものは口に入れぬ！  
这顿饭是怎么回事？！我不会把这么难吃的东西放进嘴里！ 

貴様みたいなものには、わしなど殺せぬ。  
我不可能被像你这样的人杀死。  
### 笔记
・作为惯用句的固定表达方式，表示“不~”的意思    
・古い表現であまり日常生活で使うことはない   
<b>・する ￫ せぬ</b>  
<b>・くる ￫ こぬ</b>      
<b>・いる ￫ おらぬ</b>      

## 98.〜ぬきにして・〜ぬきで
### 释义 
中 : 除去……  
日 : 〜を考慮に入れず / 〜を除いて  
### 接续
名词 + 抜きで(は)  
名词 + 抜きに(は)  
名词 + を + 抜きにして(は)  
名词 + は + 抜きとして(は)  
名词 + 抜き + の + 名词  
### 例句
冗談は抜きにして、そろそろ本題に入りましょう。  
玩笑就到此为止，是时候进入正题了。  

あいつら全然来ないな、あいつら抜きで始めちゃおうぜ。  
他们都不来呀，那就在排除他们的情况下开始吧。  

ワサビ抜きの寿司なんて寿司じゃないよ。  
没有芥末的寿司什么的不能算寿司啊。  
### 笔记
・用来表示“把一般都会包含的事物、理所当然会有的事物排除在外”之意  

## 99.〜ぬく
### 释义 
中 : 坚持……；努力到最后……  
日 : 最後まで頑張って〜する  
### 接续
动词连用形 + ぬく  
### 例句
大変かもしれないけど、最後まで頑張ってやりぬいてみせる。  
也许会很困难，但我会尽最大的努力，坚持到底。  
### 笔记
・把所必须的行为、过程做完的意思  
<b>・经受着痛苦而完成的意思较强</b>  
・某件事已完成到非常深或相当大的程度  
<b>・几乎总是用于与人类行为相关</b>  

## 100.〜の上で ・〜上の
### 释义 
中 : 从……方面  
日 : ～という点から考えると / ～という方面で  
### 接续
名词 + の + 上で  
名词 + 上の  
### 例句
カッパは想像上の生き物である。  
河童是一种虚构的生物。  

それは飽くまで計算上の話で、実際にはそう簡単にはいきません。  
这再怎么说也不过是从预测的角度来进行的，实际情况不会那么容易。  

立場上ああいう発言をしたが、個人的には争う気持ちは全くない。  
我这样说是出于我的立场，但我个人并没有争论的欲望。   
### 笔记  
・「～の上で / ～上の」在想要表达“从看到的……来判断的话，或从……的角度来考虑的话，会是怎么样的”之意时使用本句型      
・尽管表达了观点，「～の上で / ～上の」<b>不能用来表明个人的观点。</b>    
因此，它不能用作后缀用于正在考虑某事物的人员或组织的名称
