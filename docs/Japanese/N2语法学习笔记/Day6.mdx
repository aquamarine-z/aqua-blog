# N2语法学习笔记Day6

## 51.〜っぽい
### 释义
中 : 像……  
日 : ①動詞 + っぽい：〜しやすい / よく〜する  
②名詞 / い形容詞 + っぽい：～のように感じる / ～のように見える  
③名詞 + っぽい：〜が多いと感じる
### 接续
动词连用形 ＋ っぽい  
形容词词干 ＋ っぽい  
形容动词词干 ＋ っぽい  
名词 ＋ っぽい  
### 例句
今日の月は写真で撮ると白っぽいが、目で見ると赤色だ。  
今天的月亮用照片拍出来的话好像是白色的，但是用眼睛看的话是红色的。  

歳を取ったからか、最近忘れっぽくて困っている。  
可能是因为上了年纪，最近很健忘，很苦恼。  

あの子はまだ小学生なのに、とても大人っぽい。  
那孩子明明还是个小学生，却像个小大人。  
### 笔记
①动词连用形 + っぽい：有做……的倾向，表示某种倾向很强烈。  
②形容词词干/形容动词词干 + っぽい：展现存在的所有迹象  
③名词 + っぽい：尽管有迹象表明与……相像，但本质不是。并不是陈述次数之多，而是针对事物的性质而言的。  
多用于不好的事物。   

## 52.〜てでも
### 释义
中 : 不择一切手段也要……  
日 : 〜という手段をとってでも  
### 接续
动词て形 + でも  
### 例句
決勝戦なので、怪我は少し痛むけど無理してでも出場したい。  
因为是决赛，虽然受伤的地方还隐隐作痛，但即使是勉强自己，我也想出场。  

手間と時間をかけてでも、日本語がぺらぺらになりたい。  
即使需要花费大量的时间和精力，我也想变得能说一口流利的日语。  
### 笔记
由于「てでも」表达强烈的意志，因此经常出现在包含其他表示愿望或意图的短语的句子中，例如「ほしい、たい、ようと思う」  

## 53.〜て（は）ならない
### 释义
中 : ……得不得了； 不由得……    
日 : 気持ちが抑えられないほど〜だ / ひどく〜だ    
### 接续
动词て形 + ならない    
形容词て形 + ならない    
形容动词词干 + で + ならない    
### 例句
N１に合格できるかどうか心配でならない。    
我很担心能不能通过N1。    

昨日はカラオケでオールしたので、眠くてならない。    
昨天在卡拉ok玩得很尽兴，困得不得了。    
### 笔记
・「〜てならない」表示<b>情不自禁地产生某种感情或感觉，连自己都控制不了。</b>     
・除了自发和无法控制之外，<b>「〜てならない」几乎总是持续进行的，而不仅仅是短暂的。 </b>   

## 54.〜てまで・〜までして
### 释义
中 : 即使…也要…； 就连…也…    
日 : ∼さえして／∼もして  
### 接续
动词て形 + まで  
名词 + までして  
### 例句
せっかく徹夜してまで勉強したのに、結果は散々だった。  
明明都拼命熬夜学习了，结果还是不怎么样。  

株のために借金までするなんて、本当にバカだね。  
为了炒股而到借钱的地步什么的，真是太傻了。     
### 笔记
・接在一个极端的事例之后，说话人在带着某种感情来表达“到这种极端的程度”之意时，使用本句型。  
<b>后半句多是表示人的意志、主张、判断、评价等的句子。</b>  
「まで」 表示某种行为、状态或范围的“极限”或“最终点”。  
当「まで」与动作结合时，开始衍生出强调行为极限的用法。特别是通过表达一种<b>“超出常规极限的行为”，逐渐带上了即使……也要……的决心或程度</b>的语感。  

## 55.〜てみせる
### 释义
中 : 有努力做……的打算。  
日 : がんばって〜するつもりだ / 本当に〜するつもりだ
### 接续
动词て形 + みせる  
### 例句
将来は自分の会社を作って成功してみせる。  
将来一定要努力创办自己的公司。  
### 笔记
说话人强烈地想要向别人表达自己经过努力一定要做到或能做到时的表达方式。  
用通过向别人展示而达到鼓励自己的目的。  
在别人面前以实际的动作来介绍某物时也可使用本句型。   
<b>表示做某动作给对方看或表示决心、意志。  </b>  

## 56.〜というものではない・というものでもない
### 释义
中 : 不能说具备某种条件就一定……。  
日 : ある条件が整えば必ず～ということになるとは言えない。
### 接续
动词简体形 ＋ というものではない・というものでもない  
形容動詞 ＋（だ）＋ というものではない・というものでもない  
名詞 ＋（だ）＋ というものではない・というものでもない  
### 例句
教科書に書いていることが絶対的に正しいというものでもない。  
教科书上写的东西也不是绝对正确的。  

練習問題は一度やれば終わりだというものではない。間違ったところをよく復習することが大切だ。  
练习题并不是做一次就完了。重要的是要经常复习做错的地方。  

医師の仕事は資格をとればできるというものではない。常に最新の治療法を研究する姿勢がなければいけない。  
医生的工作可不是只要取得资格证书就能做的。必须得有不断研究最新医疗方法的态度。  
### 笔记
<b>并不是表述具体的事实，而是表达说话人就事物的本质发表的观点或感想。</b>  
多与表达条件的说法「～ば」等或「～からといって」连用。「～というものでもない」是稍显委婉的表达方式。    

## 57.〜といった
### 释义
中 : 接在几个例子之后，对例子进行汇总。  
日 : いくつかの例の後につけて、例をまとめる。
### 接续
名詞 + といった  
### 例句
わたしはケーキ、ポテトチップス、ハンバーガーといったカロリーの高いものが大好きなんです。  
我非常喜欢吃蛋糕、炸薯片、汉堡包等高热量的食品。  

京都とか鎌倉といった古い街には寺が多い。
像京都、镰仓这种历史悠久的城市里有很多寺庙。  

「エコ」という言葉は、「環境にいい」といった意味で使われるようになった。  
“环保”这个词，用来表示“对环境有好处”之类的意思。  
### 笔记
前接可以明确知道此外还有其他很多例子的词语。  
如例句②那样，多与「や」「とか」等搭配使用。  
也有如例句③举出一个有代表性的例子，说明大致内容的用法。  

## 58.〜といったら
### 释义
中 : 强调……的程度非同寻常。  
日 : ～の程度が普通ではないということを強調する。
### 接续
名词 + といったら  
### 例句
そのニュースを聞いたときの驚きといったら、しばらくは声も出ないほどだった。  
听到那条消息时震惊得一时说不出话来。  

締め切りの前の仕事の忙しさといったら、君には想像もできないと思うよ。  
说到临近截止日期时工作的繁忙程度，我觉得是你无法想象的。  
### 笔记
「〜といったら」 常用于感叹语句中，强调某个事物或现象的特点或程度。  
「〜といったら」 还可以用来引出具体的例子，尤其是在描述某个类别或范围内的代表性事物时。  

## 59.〜とか（で）
### 释义
中 : 据说……。  
日 : 〜そうで  
### 接续
动词简体形 + とか（で）  
形容词简体形 + とか（で）  
形容动词词干 + だ + とか（で）  
名词 + だ + とか（で）  
### 例句
田中さんは熱があるとかで、今日は休むそうです。  
据说田中发烧了，今天好像请假了。  

遠藤さんと近藤さんが結婚するとか。意外じゃない？  
听说远藤和近藤要结婚了。有点意外。  
### 笔记
表推测 为N3语法とか + で构成   

## 60.〜どころではない・〜どころか
### 释义
中 : 不是……的程度，实际上存在很大差异。  
日 : ～の程度ではなく、実際はそれとは大違いだ。
### 接续
动词简体形 + どころではない・どころか  
名词 +（な/である）+ どころではない・どころか  
形容动词词干 +（な/である）+ どころではない・どころか    
### 例句
こんな下手なチームでは、何度試合をしても一度も勝てないどころか、1点も入れられないだろう。  
这么差的球队，就算多次参加比赛，岂止一场也赢不了，甚至连1分都得不到吧。  

マナーが悪い人をちょっと注意したら、謝るどころか、逆にわたしにどなった。  
稍微提醒了一下不懂礼貌的人，对方非但没有道歉，反而大声斥责我。  
### 笔记
「～どころか」的前后项是程度相差很大或相反的事情。多用于叙述比前项更差的状况。  

## 总结
跟 と 相关的接续几乎都相同 可以同时记忆
