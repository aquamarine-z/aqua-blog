# N2语法学习笔记Day13

## 121.〜ものだから
### 释义 
中 : 就是因为……。  
日 : ～ので。  
### 接续
动词简体形＋ものだから  
形容词简体形＋ものだから  
形容动词词干＋な＋ものだから  
名词＋な＋ものだから  
### 例句
事故で電車が遅れたものですから、遅くなってすみません。  
因为交通事故电车延误，所以来晚了，对不起。  

日本の習慣を知らないものですから、失礼なことをするかもしれません。  
由于不知道日本的习俗，可能有失礼的地方。   
### 笔记
是<b>主观强调原因，理由的作用</b>，往往带有辩解的语气，常用于辩解的场合。  
口语也可以说「だって～もんだから」用于解释、辩解。  
「ことだから」相比更加主观

## 122.〜ものなら
### 释义 
中 : 如果能……的话，想这样做或想让别人这样做。  
日 : もし～できるなら、そうしたい・そうしてほしい。  
### 接续
动词可能形 + ものなら  
动词辞书形 + ものなら  
### 例句
願いが叶うものなら、あの彼と結婚したい。  
如果愿望能实现的话，我想和那个他结婚。 

戻れるものなら20年前のわたしに戻って人生をやり直したい。  
如果时光能够倒流，我想变回20年前的自己，重新活一次。  

あの日の出来事を忘れられるものなら忘れたいのに……。  
那天发生的事情如果能够忘掉的话真想忘了啊……    
### 笔记
<b>用于假设近于不可能发生的事情。</b>  
前接可能含义的动词。后续表达说话人希望和期待的句子。  

## 123.〜ものの・〜とはいうものの
### 释义 
中 : 虽然……是事实，可并未发生由此事实所理所当然联想到的事情。  
日 : ～は事実なのだが、その事実から当然想像されることが起こらない。  
### 接续
动词简体形 ＋ ものの  
形容词简体形 ＋ ものの  
形容动词词干 ＋ な/である ＋ ものの  
名词＋ である ＋ ものの  
名词＋ とはいうものの  
### 例句
高価な着物を買ったものの、着るチャンスがない。  
尽管买了昂贵的和服，却没机会穿。  

明日の天気です。関東地方は晴れるものの、風が強いため寒く感じられるでしょう。  
现在播报明日天气情况。关东地区天气晴朗，可是由于大风也许还是会感到寒冷。   
### 笔记
前接事实或可能性很高的事物。  
后续表达并不是想像的那样，还有问题存在这两种含义的句子。  
后面不接要求听话人做某事的句子。  

## 124.〜やら〜やら
### 释义 
中 : 有很多例子，比如……或其他……等。  
日 : ～や、ほかの～などの例があるように、いろいろだ。  
### 接续
名词 ＋ やら  
动词辞书形 ＋ やら  
形容词辞书形 ＋ やら  
### 例句
勝ったチームの選手たちは、泣き出すやら飛び上がるやらさまざまに喜びを表した。  
获胜球队的选手们以各种方式表达了他们的喜悦，有的喜极而泣，有的高兴得跳起来。  

だまされたとわかったときは腹が立つやら情けないやらで、気持ちを抑えることができなかった。  
得知自己被骗了的时候，又生气又觉得窝囊，无法控制自己的情绪。  

テーブルの上には、四角いものやら丸いものやらいろいろな形の皿が置いてあった。  
桌上放着方形、圆形等各种形状的盘子。  
### 笔记
罗列同一类别的词语，强调数量众多，但并非整齐划一。  
多用于列举说话人无法界定的情况以及因情况复杂而觉得辛苦的事情。  

## 125.〜ようとしている
### 释义 
中 : 在......这个变化发生的前一刻、马上就要.......。  
日 : ～という変化が起こる少し前だ・もうすぐ～する。  
### 接续
动词意志形 + としている  
### 例句
桜が満開になろうとしているとき、雪が降った。  
樱花即将完全盛开之时，下雪了。  
### 笔记
前接表示瞬间完成意义的动词（始まる・幕が開く）等。
书面语。   

## 126.〜ようものなら
### 释义 
中 : 如果……的话，就会产生严重的后果。  
日 : もし～たら、大変なことになる。  
### 接续
动词意志形 + ものなら  
### 例句
わたしはアレルギー体質なので、合わない食品を食べようものなら、体のあちこちがかゆくなる。  
我是过敏体质，所以要是吃了不合适的食物，就会浑身发痒。  

山道は危ない。ちょっと足を踏み外そうものなら、大けがをするだろう。  
山路很危险。稍一失足踏空就会受重伤吧。  
### 笔记
表示假定条件，如果这一假定成立，将导致不良的后果，  
<b>故后面往往是消极的结果。</b>  

## 127.〜を頼りに ・〜を頼り ・〜を頼りにして
### 释义 
中 : 依靠、借助……。  
日 : ～を助けに、～をたのみにして、～をたのみとして。  
### 接续
名词+を頼りに  
名词+を頼り  
名词+を頼りにして  
### 例句
レシピを頼りに、料理を作ります。  
我根据食谱来烹饪食物。  

日本のアニメは字幕を頼りにすれば、理解できます。  
借助字幕，就能看懂日本的动漫。  
### 笔记
表示凭借或依靠某人或某物来进行后项动作。

## 128.〜を問わず
### 释义 
中 : 是否……并不是问题，无论哪种情况结论都是同样的。  
日 : ～がどうかは問題ではなく、どれにも同じことが言える。   
### 接续
名词 + を問わず  
### 例句
このクレジットカードは国内、国外を問わず、いろいろな場所で使える。  
这种信用卡不管国内还是国外，很多地方都能使用。  

このマラソン大会には、性別、年齢を問わず、だれでも参加できます。  
本届马拉松大会，不分性别、年龄，谁都能参加。  
### 笔记
前接包含诸多差异的或者概括性词语（年齢・国籍・天候）等，或者反义词构成的词组（男女・内外・有無）等。  

## 129.〜をはじめ（として）
### 释义 
中 : ......是有代表性的例子，除此以外还有很多其他的。  
日 : ～が代表例で、そのほかにもいろいろある。   
### 接续
名词 + をはじめ（として）  
名词 + をはじめとする + 名词  
### 例句
日本には「桃太郎」をはじめとして、おじいさん、おばあさんが出てくる昔話が多い。  
以《桃太郎》为代表，在日本有很多关于老爷爷、老奶奶的故事。  

このあたりには、市役所をはじめとする市の公共の建物が多い。   
在这一带有许多像市政府这样的市级公共建筑物。   
### 笔记
从许多事物中列举出有代表性的事物。  
后续表达包含有代表性的例子在内的多个事物的词语。书面语。   

## 130.〜をめぐって
### 释义 
中 : 针对……进行讨论、采取对立的行为、散布谣言等。  
日 : ～について議論・対立的行為・うわさなどをする。  
### 接续
名词 + をめぐって  
名词 + をめぐる + 名词  
名词 + をめぐって + の + 名词  
### 例句
親が残した土地をめぐって親族が争っている。  
针对父母留下的土地，亲人之间展开了争夺。

消費税の問題をめぐり、国会は大きく揺れた。  
围绕消费税的问题，国会发生了大动荡。  
### 笔记
动作主体多是多人。后续表示与多人相关联的动词句。  
书面语。  

## 131.〜をもとにして
### 释义 
中 : 以……为素材、基础、根基等制作出某物。  
日 : ～という素材・基礎・土台などからあるものを作り出す。  
### 接续
名词 + をもとにして  
名词 + をもとにした + 名词  
### 例句
このシャツのマーク、面白いでしょう。「花」という漢字をもとにして作ったんだそうです。  
这件衬衫的商标很有意思吧？听说是根据汉字“花”创作的。  

あの飛行機事故をもとにした映画はこれだけじゃない。ほかにもある。  
以那次飞机事故为素材拍摄的电影不止这部。还有其他的。  
### 笔记
不用于实际的材料。  
后面多接含有表示新创作出某物、某物被创作出来等含义的动词（作る・デザインする・考え出す・できる）的句子。  

## 132.〜のだから / 〜んだから
### 释义 
中 : 因为……所以……。  
日 : ～なので。  
### 接续
形容词简体形＋のだから / んだから  
动词辞书形 + のだから / んだから  
名词＋な＋ のだから / んだから  
形容动词词干＋な＋ のだから / んだから  
### 例句
この本は難しいんだから、何回も読まないとわからないんだろう。  
这本书很难，所以不多看几遍不会明白吧。  

あなたのミスで事故が起きたのだから、あなたが責任を取ってください。  
因为你的失误发生了事故，所以请你负责。  

もう子供じゃないんだから、自分の部屋は自分で掃除しなさいよ。  
你已经不是小孩子了，自己的房间要自己打扫。  
### 笔记
表示双方都知道的理由。
