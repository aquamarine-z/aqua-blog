# 日语词语变形总结


## 1. 丁寧語(连用形+ます)
### 1.一类动词
词尾う段假名变成同行い段假名 再后面加上ます
### 2.二类动词
去掉词尾る 直接加上ます
### 3.三类动词(サ変)
去掉词尾する　直接加上します
### 4.三类动词(カ変)
来る（くる）-> 来ます（きます）
### 5.一类形容词
在词尾加上です
### 6.二类形容词
在词尾加上です
|动词类型 | 辞書形 | 丁寧語  |
|:-----|:----|:-----|
| 一类动词 | 乗る（のる）  | 乗ります |
| 二类动词 | 食べる（たべる） | 食べます |
| 三类动词 | 結婚する（けっこんする）  | 結婚します |
| 一类形容词 | 可愛い（かわいい） | 可愛いです |
| 二类形容词 | 綺麗（きれい） | 綺麗です |


## 2.丁寧語(否定)
### 1.一 二 三类动词
由连用形+ません
### 2.一 二类形容词
辞書形+(ではありません/じゃありません)

示例
|动词类型 | 辞書形 | 丁寧語(否定)  |
|:-----|:----|:-----|
| 一类动词 | 乗る（のる）  | 乗りません |
| 二类动词 | 食べる（たべる） | 食べません |
| 三类动词 | 結婚する（けっこんする）  | 結婚しません |
| 一类形容词 | 可愛い（かわいい） | 可愛いではありません |
| 二类形容词 | 綺麗（きれい） | 綺麗ではありません |

## 3.ない形(一般形否定)
### 1.一类动词
将最后一个假名由う段变为あ段 在后面加上ない
特殊情况:结尾为う的动词 需要去う变成わない
### 2.二类动词
去掉词尾る 后面加上ない
### 3.三类动词(サ変)
去掉词尾する　直接加上しない
### 4.三类动词(カ変)
来る（くる）-> 来ない（こない）
### 5.一类形容词
将最后一个假名由い变为く　在后面加上ない
### 6.二类形容词
在后面加上(ではない/じゃない)

ない形变形范例
|动词类型 | 辞書形 | ない形  |
|:-----|:----|:-----|
| 一类动词 | 知る（しる） | 知らない |
| 二类动词 | 食べる（たべる） | 食べない |
| 三类动词 | 存ずる（ぞんずる） | 存じない |
| 一类形容词 | 可愛い（かわいい） | 可愛くない |
| 二类形容词 | 綺麗（きれい） | 綺麗ではない |
## 4. て形
### 1.一类动词
根据最后一个假名的不同 分为不同的变形方式 如下表
| 结尾 | 变形方式 | 举例 |
|:-----|:----|:-----|
|す|去掉す,在后面加上して|愛す（あいす）-> 愛して|
|く/ぐ|去掉く/ぐ,在后面加上いて/いで|響く（ひびく）-> 響いて  /  泳ぐ（泳ぐ）->泳いで|
|う/つ/る|去掉结尾う段假名,在后面加上って|待つ（まつ）->待って  /  伺う（うかがう）->伺って  /  参る（まいる）->参って|
|ぬ/む/ぶ/ふ|去掉结尾う段假名,在后面加上んで|死ぬ（しぬ）->死んで  /  住む（すむ）->住んで  /  並ぶ（ならぶ）->並んで|
### 2.二类动词
去掉词尾る 后面加上て
### 3.三类动词(サ変)
去掉词尾する　直接加上して
### 4.三类动词(カ変)
来る（くる）-> 来て（きて）
### 5.一类形容词
变词尾い为く　后面加上て
### 6.二类形容词
在后面直接加上で

动词て形变形范例
|动词类型 | 辞書形 | て形  |
|:-----|:----|:-----|
| 一类动词 | 乗る（のる）  | 乗って |
| 一类动词 | 住む（すむ） | 住んで |
| 一类动词 | 泳ぐ | 泳いで |
| 二类动词 | 食べる（たべる） | 食べて |
| 三类动词 | ゲームする（game）  | ゲームして |
| 一类形容词 | 可愛い（かわいい） | 可愛くて |
| 二类形容词 | 綺麗（きれい） | 綺麗で |


## 5.た形
### 1.一类动词
根据最后一个假名的不同 分为不同的变形方式 如下表
| 结尾 | 变形方式 | 举例 |
|:-----|:----|:-----|
|す|去掉す,在后面加上した|愛す（あいす）-> 愛した|
|く/ぐ|去掉く/ぐ,在后面加上た|響く（ひびく）-> 響いた  /  泳ぐ（泳ぐ）->泳いた|
|う/つ/る|去掉结尾う段假名,在后面加上た|待つ（まつ）->待った  /  伺う（うかがう）->伺った  /  参る（まいる）->参った|
|ぬ/む/ぶ/ふ|去掉结尾う段假名,在后面加上た|死ぬ（しぬ）->死んだ  /  住む（すむ）->住んだ  /  並ぶ（ならぶ）->並んだ|
### 2.二类动词
去掉词尾る 后面加上た
### 3.三类动词(サ変)
去掉词尾する　直接加上した
### 4.三类动词(カ変)
来る（くる）-> 来た（きた）
### 5.一类形容词
变词尾い为かった
### 6.二类形容词
在后面直接加上だった

动词た形变形范例
|动词类型 | 辞書形 | た形  |
|:-----|:----|:-----|
| 一类动词 | 乗る（のる）  | 乗った |
| 一类动词 | 住む（すむ） | 住んだ |
| 一类动词 | 泳ぐ | 泳いだ |
| 一类动词 | 伺う（うかがう）  | 伺った |
| 二类动词 | 上げる（あげる）  | 上げた |
| 三类动词 | ゲームする（game）  | ゲームした |
| 一类形容词 | 可愛い（かわいい） | 可愛かった |
| 二类形容词 | 綺麗（きれい） | 綺麗だった |
